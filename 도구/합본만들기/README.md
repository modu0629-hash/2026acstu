# 수복이 합본 만들기 도구

단원별 인쇄본 파일이 source of truth. 단원 파일 수정 후 **`합본만들기.bat` 더블클릭 한 번**이면 합본 자동 갱신.

---

## 📂 파일 구조

```
2026acstu/
├─ 도구/
│  └─ 합본만들기/
│     ├─ 합본만들기.bat       ← 더블클릭 (사용자가 쓸 것)
│     ├─ merge.py             ← 메인 스크립트
│     ├─ config.json          ← 폰트 폴더 경로 등 환경 설정
│     └─ README.md            ← 이 파일
└─ 대수/
   ├─ 여러가지방정식-기본_인쇄본.html         ← raw (수정 대상)
   ├─ 연립_방정식_과_부정_방정식-기본_인쇄본.html  ← raw
   ├─ ⋯
   ├─ 2026 1학기 기말 공통수학 수복이(초판).html        ← 합본 (자동 생성/갱신)
   └─ 2026 1학기 기말 공통수학 수복이(초판).meta.json   ← 메타 (단원 순서·파일 자동 기록)
```

---

## 🔁 일상 사용법 (단원 수정 후)

1. 단원별 `.html` 파일 수정 (문제 추가, 풀이 수정 등)
2. **`합본만들기.bat` 더블클릭**
3. 끝. 합본이 자동 갱신됨

스크립트가 알아서:
- `대수/` 폴더의 `*.meta.json` 모두 자동 검색
- 각 메타에 적힌 단원 파일들 다시 읽음
- 문제 수 자동 카운트 → 1~N 연번 재부여
- 합본 한글 음절 모두 모아서 폰트 풀 서브셋 새로 생성
- 빠른정답 flex 3-col + 해설 자연 column break 처리
- 정합성·sol-id 검증

## 🆕 새 합본 만들 때 (학기/시험 별 1회)

새로운 합본 (예: 2학기 중간고사 합본)을 처음 만들 때만 config 파일 작성:

```bash
# 예: 2학기 중간고사 합본 만들기
python merge.py --init my-config.json
```

`my-config.json` 예시:
```json
{
  "title": "2026 2학기 중간 공통수학 수복이(초판)",
  "header_odd": "공통수학 1",
  "header_even": "2026 수복이",
  "base_dir": "C:\\Users\\byhah\\OneDrive\\code\\2026acstu\\대수",
  "out_dir": "C:\\Users\\byhah\\OneDrive\\code\\2026acstu\\대수",
  "base_template_file": "조합_기본_인쇄본.html",
  "units": [
    {"name": "여러가지방정식",       "file": "여러가지방정식-기본_인쇄본.html"},
    {"name": "연립방정식과 부정방정식", "file": "연립_방정식_과_부정_방정식-기본_인쇄본.html"},
    {"name": "일차부등식",           "file": "일차_부등식-기본_인쇄본.html"},
    {"name": "이차부등식",           "file": "이차_부등식-기본_인쇄본.html"},
    {"name": "경우의수",             "file": "경우의수-기본_인쇄본.html"},
    {"name": "순열",                "file": "순열-기본_인쇄본.html"},
    {"name": "조합",                "file": "조합_기본_인쇄본.html"}
  ]
}
```

만들어진 후엔 `*.meta.json`이 같이 생성되므로, 그 후엔 **`합본만들기.bat` 더블클릭만**.

## 🔧 단원 추가/제거

새 단원 파일을 추가하거나 기존 단원을 제거할 때:

1. `대수/` 폴더에 단원 .html 추가/제거
2. **합본 옆 `*.meta.json`을 텍스트 편집기로 열어** `units` 목록 수정 (한 줄 추가/제거)
3. `합본만들기.bat` 더블클릭

---

## ⚙️ config.json (환경 설정)

도구 폴더 안 `config.json`은 **환경별 설정** (폰트 풀 폴더 등). 합본별 설정은 `*.meta.json`.

```json
{
  "font_full_dir": "Z:\\HDD1\\...\\경기서체웹폰트\\woff"
}
```

다른 PC에서 작업하면 폰트 폴더 경로만 여기서 바꾸면 됨.

## 🧪 검증

`merge.py` 실행 시 자동으로:
- `pnum` 1~N 연속
- `sol-id` 1~N 정확히 1개씩
- `qa-item` 수 일치
- 단원별 추출 카운트 출력

추가로 시각·렌더링 검증을 원하면 (선택): 합본 .html을 브라우저에서 직접 열어 확인.

---

## ❓ 문제 해결

### "풀 폰트 없음" 에러
→ `config.json`의 `font_full_dir` 경로 확인. Z 드라이브에 외장 디스크가 마운트되어 있어야 함.

### "메타 파일 없음" 에러
→ 새 합본은 먼저 `python merge.py --init <config.json>` 실행 필요.

### 글리프 fallback (한글 깨짐)
→ 단원에 새로운 한자/특수문자 추가 시 자동으로 풀 서브셋 다시 생성하므로 정상 작동해야 함. 안 되면 폰트 풀 폴더에 풀 woff(2~3MB)가 있는지 확인.

조합

조합

서로 다른 $n$개에서 순서를 생각하지 않고 $r$개$(0 \le r \le n)$를 택하는 것을 $n$개에서 $r$개를 택하는 조합이라 하고, 이 조합의 수를 기호 $_n\mathrm{C}_r$로 나타낸다.

$_n\mathrm{C}_r = \dfrac{_n\mathrm{P}_r}{r!} = \dfrac{n!}{r!\,(n-r)!}$    (단, $0 \le r \le n$)
$_n\mathrm{C}_r$의 C는 조합을 뜻하는 Combination의 첫 글자이다. 서로 다른 것에서 순서를 생각하지 않고 택하는 것은 조합이고, 순서를 생각하여 택하는 것은 순열이다.
예) $_7\mathrm{C}_3 = \dfrac{7!}{3!(7-3)!} = \dfrac{7!}{3!\cdot4!} = \dfrac{7\times6\times5}{3\times2\times1} = 35$
▷ 공식 유도 증명

서로 다른 $n$개에서 $r$개를 순서 있게 나열하는 방법의 수는 $_{n}\mathrm{P}_{r}$이다.

이 $_{n}\mathrm{P}_{r}$가지 각각은, 같은 $r$개를 택한 뒤 순서만 바꿔도 서로 다르게 센다. 그런데 조합에서는 순서를 구별하지 않으므로, 같은 $r$개를 택한 경우들($r!$가지)을 하나로 묶어야 한다.

따라서   $_n\mathrm{C}_r = \dfrac{_n\mathrm{P}_r}{r!} = \dfrac{n!/(n-r)!}{r!} = \dfrac{n!}{r!\,(n-r)!}$

조합의 수의 성질

①  $_n\mathrm{C}_0 = {}_n\mathrm{C}_n = 1$

$_n\mathrm{C}_0 = \dfrac{n!}{0!\cdot n!} = 1$    ($0! = 1$ 이용)     $_n\mathrm{C}_n = \dfrac{n!}{n!\cdot 0!} = 1$

②  $_n\mathrm{C}_r = {}_n\mathrm{C}_{n-r}$   (단, $0 \le r \le n$)

조합론적 증명: $n$개 중 $r$개를 선택하는 것은, 선택하지 않을 $n-r$개를 정하는 것과 일대일 대응이므로 $_n\mathrm{C}_r = {}_n\mathrm{C}_{n-r}$.

식으로 증명:   $_n\mathrm{C}_{n-r} = \dfrac{n!}{(n-r)!\cdot(n-(n-r))!} = \dfrac{n!}{(n-r)!\cdot r!} = {}_n\mathrm{C}_r$

$_n\mathrm{C}_r$의 값을 구할 때, $r > n-r$ 이면 $_n\mathrm{C}_r$을 $_n\mathrm{C}_{n-r}$로 바꾸어 계산하면 편리하다.
예)   $_7\mathrm{C}_5 = {}_7\mathrm{C}_2 = \dfrac{7\times6}{2\times1} = 21$

③  $_n\mathrm{C}_r = {}_{n-1}\mathrm{C}_{r-1} + {}_{n-1}\mathrm{C}_r$   (단, $1 \le r < n$)   — 파스칼의 법칙

조합론적 증명: 집합 $\{1, 2, \ldots, n\}$에서 $r$개를 택하는 경우를 원소 $n$의 포함 여부로 분류한다.

• $n$을 포함하는 경우: 나머지 $n-1$개에서 $r-1$개 선택 $\Rightarrow$ $_{n-1}\mathrm{C}_{r-1}$가지

• $n$을 포함하지 않는 경우: $n-1$개에서 $r$개 선택 $\Rightarrow$ $_{n-1}\mathrm{C}_r$가지

두 경우는 서로 배반이므로 $_n\mathrm{C}_r = {}_{n-1}\mathrm{C}_{r-1} + {}_{n-1}\mathrm{C}_r$.

식으로 증명:

$_{n-1}\mathrm{C}_{r-1} + {}_{n-1}\mathrm{C}_r = \dfrac{(n-1)!}{(r-1)!(n-r)!} + \dfrac{(n-1)!}{r!\,(n-r-1)!}$

$= \dfrac{(n-1)!\cdot r}{r!(n-r)!} + \dfrac{(n-1)!\cdot(n-r)}{r!(n-r)!} = \dfrac{(n-1)!\,(r+n-r)}{r!(n-r)!} = \dfrac{n!}{r!(n-r)!} = {}_n\mathrm{C}_r$

예)   $_{n-1}\mathrm{C}_{r-1} + {}_{n-1}\mathrm{C}_r$는 파스칼의 삼각형에서 인접한 두 수의 합이 아래 수임을 나타낸다.

도형의 개수

(1) 직선의 개수  — 어느 세 점도 일직선 위에 있지 않은 서로 다른 $n$개의 점에서 두 점을 이어 만들 수 있는 직선의 개수

$_n\mathrm{C}_2 = \dfrac{n(n-1)}{2}$
두 점을 지나는 직선은 오직 하나이므로 직선의 개수는 두 점을 택하는 방법의 수이고, 일직선(동일 직선) 위에 있는 모든 점들을 이으면 직선은 한 개 생긴다.

(2) 삼각형의 개수  — 어느 세 점도 일직선 위에 있지 않은 서로 다른 $n$개의 점에서 세 점을 이어 만들 수 있는 삼각형의 개수

$_n\mathrm{C}_3 = \dfrac{n(n-1)(n-2)}{6}$
일직선 위에 있는 세 개 이상의 점으로는 삼각형을 만들 수 없다.

(3) 평행사변형의 개수  — $m$개의 평행선과 $n$개의 평행선이 서로 만날 때, 만들어지는 평행사변형의 개수

$_m\mathrm{C}_2 \,\times\, {}_n\mathrm{C}_2$

보충: 같은 직선 위에 점이 있는 경우

$n$개의 점 중 $k$개가 한 직선 $\ell$ 위에 있고, 나머지 점 중 어떤 $3$점도 일직선이 아닐 때:

직선의 수  $= \;{}_n\mathrm{C}_2 - {}_k\mathrm{C}_2 + 1$

삼각형의 수  $= \;{}_n\mathrm{C}_3 - {}_k\mathrm{C}_3$

$\ell$ 위의 $k$개의 점에서 만들어지는 직선 ${}_k\mathrm{C}_2$개가 실제로는 $\ell$ 하나이므로 $({}_k\mathrm{C}_2 - 1)$을 뺀다. 삼각형은 $\ell$ 위의 세 점을 택하면 일직선이므로 불가. ${}_k\mathrm{C}_3$을 뺀다.
예) $5$개의 점 중 $3$개가 한 직선 위에 있을 때:
직선의 수 $= {}_5\mathrm{C}_2 - {}_3\mathrm{C}_2 + 1 = 10 - 3 + 1 = 8$,   삼각형의 수 $= {}_5\mathrm{C}_3 - {}_3\mathrm{C}_3 = 10 - 1 = 9$

보충 — 원 위의 등간격 점에서 삼각형 개수

원 위에 $n$개의 점이 같은 간격으로 있을 때, 3개를 이어 만드는 삼각형의 종류별 개수를 구한다.

핵심 아이디어: 선택된 세 점이 원을 세 호로 나눌 때, 각 호의 점 간격 수를 $a, b, c$라 하면 $a+b+c=n$ ($a,b,c\ge1$). 원주각의 성질에 의해:

$\max(a,b,c) < \dfrac{n}{2}$  $\Longrightarrow$  예각삼각형

$\max(a,b,c) = \dfrac{n}{2}$  $\Longrightarrow$  직각삼각형   ($n$이 짝수일 때만 가능)

$\max(a,b,c) > \dfrac{n}{2}$  $\Longrightarrow$  둔각삼각형

공식 (총 삼각형 = $_n\mathrm{C}_3$)

① $n$이 홀수 ($n = 2m+1$)일 때:

직각삼각형: $\;0$

예각삼각형: $\;\dfrac{n(n-1)(n+1)}{24} = \dfrac{n(n^2-1)}{24}$

둔각삼각형: $\;\dfrac{n(n-1)(n-3)}{8}$

② $n$이 짝수 ($n = 2m$)일 때:

직각삼각형: $\;\dfrac{n(n-2)}{2}$

예각삼각형: $\;\dfrac{n(n-2)(n-4)}{24}$

둔각삼각형: $\;\dfrac{n(n-2)(n-4)}{8}$

▷ 증명 (핵심만)

직각삼각형 ($n=2m$): 지름이 되는 쌍이 $m$개. 각 지름에 대해 나머지 $n-2$개의 점이 직각꼭짓점이 되므로 $m(n-2) = \dfrac{n(n-2)}{2}$개.

예각삼각형 ($n=2m+1$): $a+b+c=n$이고 각 $\le m$인 순서 있는 쌍의 수를 세면, 유효 순열 수는 $\dfrac{n(m-1)m}{2} \times \dfrac{1}{\text{...}}$... 결과적으로 $\dfrac{n(n^2-1)}{24}$ ($n=3$: $1$개, $n=5$: $5$개, $n=7$: $14$개로 검증).

둔각삼각형: 전체 $-$ 직각 $-$ 예각으로 구한다.

검증 ($n=6$): 직각: $\dfrac{6\times4}{2}=12$, 예각: $\dfrac{6\times4\times2}{24}=2$, 둔각: $\dfrac{6\times4\times2}{8}=6$, 합계 $= 20 = {}_6\mathrm{C}_3$ ✓

1.
[증명]

(1) 조합의 식 이용:

$r \times {}_n\mathrm{C}_r = r \times \dfrac{n!}{r!\,(n-r)!} = \dfrac{n!}{(r-1)!\,(n-r)!}$

$= n \times \dfrac{(n-1)!}{(r-1)!\,(n-r)!} = n \times {}_{n-1}\mathrm{C}_{r-1}$

(2) 조합의 개념 이용:

좌변 ($r \times {}_n\mathrm{C}_r$): $n$개에서 $r$개를 택한 후, 그 $r$개 중에서 대표 $1$명을 정하는 방법의 수.

우변 ($n \times {}_{n-1}\mathrm{C}_{r-1}$): 먼저 $n$개 중에서 대표 $1$명을 정하고 ($n$가지), 나머지 $n-1$명 중에서 $r-1$명을 택하는 방법의 수.

양쪽 모두 "$n$명 중 $r$명을 뽑고 그 중 대표 $1$명을 정하는 방법"을 세는 것이므로 $r \times {}_n\mathrm{C}_r = n \times {}_{n-1}\mathrm{C}_{r-1}$.

(3) $n$명이 악수한 총 횟수 $= {}_n\mathrm{C}_2 = \dfrac{n(n-1)}{2} = 45$.   $n(n-1)=90=10\times9$ 이므로 $n=10$.

(1) 조건 $a_2 > a_3 > a_5$: 위치 $\{2,3,5\}$에 들어갈 $3$개 선택 → $_6\mathrm{C}_3=20$가지, 뽑힌 $3$수는 크기 순서 고정(1가지).

조건 $a_1 > a_4$: 나머지 $3$개 중 위치 $\{1,4\}$에 들어갈 $2$개 선택 → $_3\mathrm{C}_2=3$가지, 큰 수가 $a_1$(1가지).

나머지 $1$개는 위치 $6$에 자동 배정.   ∴ $20 \times 3 = 60$

(2) 백의 자리 $a \in \{5,6\}$ ($500 < \overline{abc} < 700$이므로).

$a=5$: $5>b>c$인 자연수 $b,c$ 선택 → $\{1,2,3,4\}$에서 $2$개 선택: $_4\mathrm{C}_2=6$

$a=6$: $6>b>c$인 자연수 $b,c$ 선택 → $\{1,2,3,4,5\}$에서 $2$개 선택: $_5\mathrm{C}_2=10$

∴ $6+10=16$

8.
[정답]  $24$

순서(독창·중창·합창)가 고정된 $1$부와 $2$부에 팀을 배정한다.

$1$부 독창: 독창 $2$팀 중 $1$팀 → $2$가지

$1$부 중창: 중창 $2$팀 중 $1$팀 → $2$가지

$1$부 합창: 합창 $3$팀 중 $1$팀 → $3$가지

$2$부 독창·중창: 나머지 독창·중창 $1$팀씩 자동 → $1$가지

$2$부 합창 $2$팀 순서: 나머지 합창 $2$팀 배열 → $2!=2$가지

∴ $2\times2\times3\times1\times2=24$

[정답]   (1) 직선 $9$, 삼각형 $17$    (2) 직선 $10$, 삼각형 $10$    (3) 직선 $20$, 삼각형 $100$

(1) 삼각형 도형 (꼭짓점 $3$개 + 각 변 위 점 $1$개씩 = 총 $6$점, 각 변 위 $3$점 일직선):

전체 점 $6$개이고, 세 변 위에 각각 $3$점씩 일직선.

직선: $_6\mathrm{C}_2 - 3\times(_3\mathrm{C}_2-1) = 15 - 3\times2 = 9$

삼각형: $_6\mathrm{C}_3 - 3\times{}_3\mathrm{C}_3 = 20 - 3\times1 = 17$

(2) 반원 도형 (지름 양 끝 $2$점 + 호 위 $3$점 = 총 $5$점, 어느 $3$점도 일직선 없음):

직선: $_5\mathrm{C}_2 = 10$     삼각형: $_5\mathrm{C}_3 = 10$

(3) 오각별 도형 (꼭짓점 $5$개 + 내부 교점 $5$개 = 총 $10$점, 별의 각 선분에 $4$점씩 일직선 $5$개):

직선: $_{10}\mathrm{C}_2 - 5\times({}_4\mathrm{C}_2-1) = 45 - 5\times5 = 45-25 = 20$

삼각형: $_{10}\mathrm{C}_3 - 5\times{}_4\mathrm{C}_3 = 120 - 5\times4 = 120-20 = 100$

10.
[정답]  (1) $85$   (2) $54$

(1) 왼쪽 정렬 계단 (위→아래 $3,3,3,5,5$칸, 총 $19$칸):

두 행 경계 쌍 $(r_1, r_2)$에 대해 구간 최소 폭이 $m$이면 가로 경계쌍 수 $\binom{m+1}{2}$.

최소 폭 $3$인 쌍 $12$개 (행 $4,5$만 포함하지 않는 경우): 각 $\binom{4}{2}=6$ $\Rightarrow$ $72$.

최소 폭 $5$인 쌍 $3$개 (행 $4,5$만 포함): 각 $\binom{6}{2}=15$ $\Rightarrow$ $45$.

전체 $= 72+45=117$.

정사각형: $1\!\times\!1=19$, $2\!\times\!2=2+2+2+4=10$, $3\!\times\!3=1+1+1=3$. 합 $=32$.

$\therefore$ 정사각형 아닌 직사각형 $= 117-32=85$.

(2) 오른쪽 정렬 계단 (위→아래 $3,5,7$칸, 총 $15$칸):

각 행의 왼쪽 경계: 행 $1$은 $c_1\geq 4$, 행 $2$는 $c_1\geq 2$, 행 $3$은 $c_1\geq 0$. 모든 행 오른쪽 경계 $=7$.

$(r_1,r_2)$별 가로경계쌍 수:

$(0,1),(0,2),(0,3)$: $c_1\geq 4$ $\Rightarrow$ 각 $\binom{4}{2}=6$, 소계 $18$.

$(1,2),(1,3)$: $c_1\geq 2$ $\Rightarrow$ 각 $\binom{6}{2}=15$, 소계 $30$.

$(2,3)$: $c_1\geq 0$ $\Rightarrow$ $\binom{8}{2}=28$.

전체 $= 18+30+28=76$.

정사각형: $1\!\times\!1=15$, $2\!\times\!2=2+4=6$, $3\!\times\!3=1$. 합 $=22$.

$\therefore$ 정사각형 아닌 직사각형 $= 76-22=54$.

1. 삼차방정식의 풀이

삼차방정식 $ax^3+bx^2+cx+d=0$ ($a\neq 0$)은 좌변을 일차 인수와 이차 인수의 곱으로 분해하여 푼다. 일차 인수는 인수정리·조립제법으로, 남은 이차식은 인수분해 또는 근의 공식으로 처리한다.

1) 풀이 절차
❶ 정수근 후보: $\dfrac{(\text{상수항 } d \text{의 약수})}{(\text{최고차항 } a \text{의 약수})}$ 의 양·음 모두
❷ 후보 $\alpha$를 대입해 $f(\alpha)=0$인 값을 찾는다 $\Rightarrow$ $f(x) = (x-\alpha)Q(x)$
❸ 조립제법으로 이차식 $Q(x)$를 얻고, $Q(x)=0$을 풀어 나머지 두 근을 구한다
2) 인수분해 공식 활용

아래 공식을 알아두면 정수근 탐색 없이 바로 분해 가능하다.

① $x^3 \pm a^3 = (x \pm a)(x^2 \mp ax + a^2)$   (복부호 동순)
② $x^3 \pm 3ax^2 + 3a^2 x \pm a^3 = (x \pm a)^3$
③ $a^3 + b^3 + c^3 - 3abc = (a+b+c)(a^2+b^2+c^2-ab-bc-ca)$
$f(x)$의 모든 계수의 합이 $0$이면 $x=1$이, 부호 교대 합이 $0$이면 $x=-1$이 근이다. 후보 검증의 첫 단추로 활용.

2. 삼차방정식의 근과 계수의 관계

삼차방정식 $ax^3+bx^2+cx+d=0$ ($a\neq 0$)의 세 근을 $\alpha,\,\beta,\,\gamma$라 하면 다음이 성립한다.

$\alpha+\beta+\gamma = -\dfrac{b}{a},\quad \alpha\beta+\beta\gamma+\gamma\alpha = \dfrac{c}{a},\quad \alpha\beta\gamma = -\dfrac{d}{a}$
(증명) $a(x-\alpha)(x-\beta)(x-\gamma) = ax^3 - a(\alpha+\beta+\gamma)x^2 + a(\alpha\beta+\beta\gamma+\gamma\alpha)x - a\alpha\beta\gamma.$ 우변 $ax^3+bx^2+cx+d$와 계수 비교.
대칭식 변환: $\alpha^2+\beta^2+\gamma^2 = (\alpha+\beta+\gamma)^2 - 2(\alpha\beta+\beta\gamma+\gamma\alpha)$,   $\dfrac{1}{\alpha}+\dfrac{1}{\beta}+\dfrac{1}{\gamma} = \dfrac{\alpha\beta+\beta\gamma+\gamma\alpha}{\alpha\beta\gamma}$,   $\alpha^3+\beta^3+\gamma^3 - 3\alpha\beta\gamma = (\alpha+\beta+\gamma)(\alpha^2+\beta^2+\gamma^2 - \alpha\beta-\beta\gamma-\gamma\alpha)$.

3. 세 수를 근으로 갖는 삼차방정식의 작성

$\alpha,\,\beta,\,\gamma$를 세 근으로 갖고 $x^3$의 계수가 $1$인 삼차방정식은 다음과 같다.

$x^3 - (\alpha+\beta+\gamma)x^2 + (\alpha\beta+\beta\gamma+\gamma\alpha)x - \alpha\beta\gamma = 0$
즉 $(x-\alpha)(x-\beta)(x-\gamma)=0$. $x^3$의 계수를 $k\,(\neq 0)$로 만들고 싶으면 양변에 $k$를 곱한다.

4. 삼차방정식의 켤레근의 성질

1) 무리수 켤레근

계수가 유리수인 삼차방정식이 무리수 $p+q\sqrt{m}$ ($p,\,q$ 유리수, $\sqrt{m}$ 무리수, $q\neq 0$)을 근으로 가지면, 켤레 무리수 $p-q\sqrt{m}$도 근이다.

2) 허수 켤레근

계수가 실수인 삼차방정식이 허수 $p+qi$ ($p,\,q$ 실수, $q\neq 0$)을 근으로 가지면, 켤레 허수 $p-qi$도 근이다.

사차·오차 등 일반 다항방정식에서도 동일하게 성립. 따라서 실계수 다항방정식은 허근이 항상 짝수 개.

5. 삼차방정식의 허근의 성질

1) 방정식 $x^3=1$의 허근의 성질

방정식 $x^3=1$의 한 허근, 즉 $x^2+x+1=0$의 한 근을 $\omega$라 하면 다음이 성립한다. (단, $\bar{\omega}$는 $\omega$의 켤레복소수)

① $\omega^3 = 1,\ \ \omega^2+\omega+1=0$
② $\omega+\bar{\omega} = -1,\ \ \omega\bar{\omega} = 1$
③ $\bar{\omega} = \omega^2 = \dfrac{1}{\omega}$
(증명) $x^3-1=(x-1)(x^2+x+1)=0$ $\Rightarrow$ $x=1$ 또는 $x^2+x+1=0$. 허근 $\omega$는 $x^2+x+1=0$의 근 $\Rightarrow$ $\omega^2+\omega+1=0$ ⋯㉠. 켤레근의 성질로 다른 허근은 $\bar{\omega}$. 근과 계수의 관계로 $\omega+\bar{\omega}=-1,\ \omega\bar{\omega}=1$ ⋯㉡. ㉠에서 $\omega^2=-\omega-1$, ㉡에서 $-\omega-1=\bar{\omega}$이므로 $\omega^2=\bar{\omega}$. 또 ㉡에서 $\bar{\omega}=\dfrac{1}{\omega}$.

2) 방정식 $x^3=-1$의 허근의 성질

방정식 $x^3=-1$의 한 허근, 즉 $x^2-x+1=0$의 한 근을 $\omega$라 하면 다음이 성립한다.

① $\omega^3 = -1,\ \ \omega^2-\omega+1=0$
② $\omega+\bar{\omega} = 1,\ \ \omega\bar{\omega} = 1$
③ $\bar{\omega} = -\omega^2 = \dfrac{1}{\omega}$
(증명) $x^3+1=(x+1)(x^2-x+1)=0$ $\Rightarrow$ $x=-1$ 또는 $x^2-x+1=0$. 허근 $\omega$는 $\omega^2-\omega+1=0$ ⋯㉠. 켤레근으로 $\bar{\omega}$도 근. 근과 계수: $\omega+\bar{\omega}=1,\ \omega\bar{\omega}=1$ ⋯㉡. ㉠에서 $\omega^2=\omega-1$, ㉡에서 $\omega-1=-\bar{\omega}$이므로 $\omega^2=-\bar{\omega}$, 즉 $\bar{\omega}=-\omega^2$. 또 ㉡에서 $\bar{\omega}=\dfrac{1}{\omega}$.
3) [참고] 방정식 $x^3=8$의 허근의 성질

같은 방법으로 $x^3=8$의 한 허근, 즉 $x^2+2x+4=0$의 한 근을 $\omega$라 하면 다음이 성립한다.

① $\omega^3 = 8,\ \ \omega^2+2\omega+4=0$
② $\omega+\bar{\omega} = -2,\ \ \omega\bar{\omega} = 4$
③ $\bar{\omega} = -\omega-2 = \dfrac{\omega^2}{2} = \dfrac{4}{\omega}$
(증명 약식) $x^3-8=(x-2)(x^2+2x+4)=0$. 허근 $\omega$는 $\omega^2+2\omega+4=0$. 근과 계수: $\omega+\bar{\omega}=-2,\ \omega\bar{\omega}=4$. $\omega^2=-2\omega-4=2(-\omega-2)=2\bar{\omega}$이므로 $\bar{\omega}=\dfrac{\omega^2}{2}$. 또 $\omega\bar{\omega}=4$에서 $\bar{\omega}=\dfrac{4}{\omega}$.

6. 사차방정식의 풀이

사차방정식 $ax^4+bx^3+cx^2+dx+e=0$ ($a\neq 0$)도 좌변을 (일차)·(이차)·(이차)·(일차) 등의 인수곱으로 분해해서 푼다.

❶ 정수근 후보를 인수정리로 두 번 적용 $\Rightarrow$ 일차 인수 두 개 분리, 남는 이차식 풀이
❷ 좌변이 인수분해 공식($A^2-B^2$, $A^3 \pm B^3$ 등) 형태이면 즉시 분해
❸ 특수 형태(복이차·상반방정식)는 적절한 치환으로 이차방정식화 (§7, §8)
사차에서도 모든 계수의 합이 $0$이면 $x=1$이, 부호 교대 합이 $0$이면 $x=-1$이 근. 정수근 탐색의 첫 단추.

7. 복이차방정식

홀수 차수 항이 없는 사차방정식 $ax^4+bx^2+c=0$ ($a\neq 0$)을 복이차방정식이라 한다. 두 가지 풀이가 표준.

1) $t = x^2$ 치환

$t$에 대한 이차방정식 $at^2+bt+c=0$의 근 $t$를 구한 뒤 $x^2=t$의 근 $x$를 구한다.

2) $A^2-B^2$ 형 인수분해

좌변을 완전제곱으로 만들고 부족한 항을 빼서 $A^2-B^2 = (A+B)(A-B)$ 형태로 만든다. ($t$ 치환의 판별식이 음일 때 특히 유용.)

예) $x^4+x^2+1 = (x^2+1)^2 - x^2 = (x^2+x+1)(x^2-x+1)$.   $x^4-7x^2+9 = (x^2-3)^2 - x^2 = (x^2+x-3)(x^2-x-3)$.

8. 상반방정식의 풀이

계수가 좌우 대칭(또는 부호만 반대)인 사차방정식을 상반방정식이라 한다. 양변을 $x^2\,(\neq 0)$로 나눈 뒤 $x \pm \dfrac{1}{x}$를 한 변수로 치환해 이차방정식으로 만든다.

1) 대칭형 상반방정식

형태: $ax^4+bx^3+cx^2+bx+a=0$ ($a\neq 0$). 양변에 같은 계수 $a,\,b$가 좌우 대칭으로 등장.

❶ 양변을 $x^2$으로 나누면  $a\!\left(x^2+\dfrac{1}{x^2}\right) + b\!\left(x+\dfrac{1}{x}\right) + c = 0$
❷ $t = x + \dfrac{1}{x}$ 치환 $\Rightarrow$ $x^2 + \dfrac{1}{x^2} = t^2 - 2$
❸ $a(t^2-2) + bt + c = 0$, 즉 $at^2 + bt + (c-2a) = 0$   (이차방정식)
❹ $t$의 각 값에 대해 $x + \dfrac{1}{x} = t$, 즉 $x^2 - tx + 1 = 0$ 풀이
2) 반대칭형 상반방정식

형태: $ax^4+bx^3+cx^2-bx+a=0$ ($a\neq 0,\,b\neq 0$). $x^3,\,x^1$ 계수의 부호가 반대.

❶ 양변을 $x^2$으로 나누면  $a\!\left(x^2+\dfrac{1}{x^2}\right) + b\!\left(x-\dfrac{1}{x}\right) + c = 0$
❷ $t = x - \dfrac{1}{x}$ 치환 $\Rightarrow$ $x^2 + \dfrac{1}{x^2} = t^2 + 2$
❸ $a(t^2+2) + bt + c = 0$, 즉 $at^2 + bt + (c+2a) = 0$   (이차방정식)
❹ $t$의 각 값에 대해 $x - \dfrac{1}{x} = t$, 즉 $x^2 - tx - 1 = 0$ 풀이
치환식의 부호에 주의: 대칭형은 $\bigl(x+\dfrac{1}{x}\bigr)^2 = x^2 + 2 + \dfrac{1}{x^2}$, 반대칭형은 $\bigl(x-\dfrac{1}{x}\bigr)^2 = x^2 - 2 + \dfrac{1}{x^2}$.

1.
[정답] (1) $x=-1,\,1,\,2$   (2) $x=-2,\,1\pm\sqrt{3}\,i$   (3) $x=1$ (삼중근)

(1) $f(x)=x^3-2x^2-x+2$. 모든 계수의 합 $1-2-1+2=0$이므로 $x=1$이 근. 조립제법으로 $f(x)=(x-1)(x^2-x-2)=(x-1)(x-2)(x+1)$. $\Rightarrow x=-1,\,1,\,2.$

(2) 인수분해 공식 $x^3+a^3=(x+a)(x^2-ax+a^2)$ 에 $a=2$ 적용: $x^3+8=(x+2)(x^2-2x+4)=0.$ $x^2-2x+4=0$에서 $x=1\pm\sqrt{3}\,i.$

(3) 인수분해 공식 $x^3-3a x^2+3a^2 x - a^3=(x-a)^3$ 에 $a=1$: $(x-1)^3=0 \Rightarrow x=1$ (삼중근).

2.
[정답] (1) $a=-4,\,b=1$   (2) $x=2$

근과 계수의 관계 활용. 세 근을 $-1,\,3,\,r$이라 하자.

근의 곱: $\alpha\beta\gamma = (-1)\cdot 3 \cdot r = -3r = -\dfrac{6}{1} = -6 \Rightarrow r=2.$   $\therefore$ 나머지 한 근 $x=2.$

근의 합: $\alpha+\beta+\gamma = -1+3+2 = 4 = -a \Rightarrow a=-4.$

두 곱의 합: $\alpha\beta+\beta\gamma+\gamma\alpha = (-1)(3) + (3)(2) + (2)(-1) = -3+6-2 = 1 = b.$   $\therefore a=-4,\,b=1.$

3.
[정답] (1) $-1$   (2) $\dfrac{5}{7}$   (3) $3$

근과 계수의 관계: $\alpha+\beta+\gamma = 3,\ \ \alpha\beta+\beta\gamma+\gamma\alpha = 5,\ \ \alpha\beta\gamma = 7.$

(1) $\alpha^2+\beta^2+\gamma^2 = (\alpha+\beta+\gamma)^2 - 2(\alpha\beta+\beta\gamma+\gamma\alpha) = 9 - 10 = -1.$

(2) $\dfrac{1}{\alpha}+\dfrac{1}{\beta}+\dfrac{1}{\gamma} = \dfrac{\alpha\beta+\beta\gamma+\gamma\alpha}{\alpha\beta\gamma} = \dfrac{5}{7}.$

(3) 항등식 $\alpha^3+\beta^3+\gamma^3 - 3\alpha\beta\gamma = (\alpha+\beta+\gamma)(\alpha^2+\beta^2+\gamma^2 - \alpha\beta-\beta\gamma-\gamma\alpha) = 3 \cdot (-1 - 5) = -18.$ $\therefore \alpha^3+\beta^3+\gamma^3 = -18 + 3 \cdot 7 = 3.$

4.
[정답] (1) $x^3-7x+6=0$   (2) $x^3+2px^2+4qx+8r=0$

(1) 세 근 $1,\,2,\,-3$의 합 $= 0$, 두 곱의 합 $= 1\cdot 2 + 2\cdot(-3) + (-3)\cdot 1 = -7$, 세 곱 $= -6.$ $x^3$ 계수 $1$인 식: $x^3 - 0\cdot x^2 + (-7)x - (-6) = x^3 - 7x + 6 = 0.$

(2) 세 근이 $\alpha,\,\beta,\,\gamma$이면 $\alpha+\beta+\gamma=-p,\ \alpha\beta+\beta\gamma+\gamma\alpha=q,\ \alpha\beta\gamma=-r.$ 새 근 $2\alpha,\,2\beta,\,2\gamma$에 대해 합 $=2(\alpha+\beta+\gamma)=-2p$, 두 곱의 합 $= 4(\alpha\beta+\beta\gamma+\gamma\alpha)=4q$, 세 곱 $=8\alpha\beta\gamma=-8r.$

$\therefore$ 새 방정식 $x^3 - (-2p)x^2 + 4q\, x - (-8r) = x^3 + 2p\,x^2 + 4q\,x + 8r = 0.$

[정답] $a=1,\,b=-7,\,c=-3$

계수가 유리수이고 한 근이 무리수 $1+\sqrt{2}$이므로 켤레근의 성질에 의해 $1-\sqrt{2}$도 근. 세 번째 근은 주어진 $-3.$ 즉 세 근: $1+\sqrt{2},\,1-\sqrt{2},\,-3.$

근과 계수의 관계: $\alpha+\beta+\gamma = (1+\sqrt{2})+(1-\sqrt{2})+(-3) = -1 = -a \Rightarrow a=1.$

$\alpha\beta+\beta\gamma+\gamma\alpha = (1+\sqrt{2})(1-\sqrt{2}) + (1-\sqrt{2})(-3) + (-3)(1+\sqrt{2}) = -1 - 3 + 3\sqrt{2} - 3 - 3\sqrt{2} = -7 = b.$

$\alpha\beta\gamma = (-1)(-3) = 3 = -c \Rightarrow c=-3.$   $\therefore a=1,\,b=-7,\,c=-3.$

계수가 실수이고 한 근이 허수 $1-2i$이므로 켤레 $1+2i$도 근. 세 번째 근을 실수 $r$이라 하자.

근의 곱: $(1-2i)(1+2i)\cdot r = (1+4)\cdot r = 5r = -10 \Rightarrow r=-2.$   $\therefore$ 나머지 두 근: $1+2i,\,-2.$

근의 합: $(1-2i)+(1+2i)+(-2)=0=-a \Rightarrow a=0.$

두 곱의 합: $5 + (1+2i)(-2) + (-2)(1-2i) = 5 - 2 - 4i - 2 + 4i = 1 = b.$   $\therefore a=0,\,b=1.$

### Phase A 초안
- 행렬-기본 베이스 복사·치환 (챕터제목 `지수`, 헤더 `수학 · 대수`)

- 마플 자료는 그대로 베끼지 않고 변형/재작성 (저작권)

### 1차 수정 (3건)
1. 개념 끝 → 관련 문제 끼움 (행렬 패턴)
2. 모든 분수 `\dfrac` 통일
3. SVG 그래프 2개 추가 (n 짝수 / n 홀수)

### 2차 수정 (3건)
4. 슬래시 분수(`a/b`) → `\dfrac{a}{b}` 일괄 (115 LaTeX 블록)
5. SVG foreignObject 라벨 폰트 = 본문 11pt 일치
6. 두 번째 그래프 곡선 = 정확한 y=x³ 형태 (17점 polyline)

### 3차 수정 (5건, 최신)
7. y=x³ 폭 좁힘: math x `[-2,2]` → `[-1.5,1.5]`, svg x `40~240` → `80~200`
8. 두 그래프 거리 넓힘: `.fig-row gap: 6mm → 14mm`
9. 첫 그래프 `y=x^n` → `y=x²` 구체화 (17점 polyline)
10. q1 보기 5줄 강제: `.choices.five-rows { grid-template-columns:1fr }` 신규
11. 지수 안 분수 50%: CSS `mjx-script/mover/munder mjx-mfrac { font-size:50% !important }` 신규

## 학원에서 이어서 작업할 때 참고

### 절대 손대지 말 것 (행렬 베이스에서 검증된 부분)
- `<style>` 안 행렬 인쇄본 베이스 CSS 수치 (--line-h, --pad-x, .sheet 등)
- 자동 플로우 JS 5함수 (`autoWrapChoices`, `compactAndSplitConcepts`, `flowSolutions`, `rebalanceParity`, `renumberFooters`)
- JS 안 헤더 문자열 `'수학 · 대수'` / `'2026 수복이'` 두 곳
- `data-qa-id="qN"` ↔ `id="sol-N"` 1:1 매칭 (q1~q12)

### 추가만 OK
- 새 CSS 룰 (style 끝부분 추가)
- 새 SVG (foreignObject 라벨 11pt + 신명조)
- 새 .concept / .problem-block (단, q번호 충돌 주의)

### 알려진 결정 사항
- 분수: 위/아래 형태(`\dfrac`) 일관 사용. 슬래시 표기 금지.
- 지수 안 분수: 본문 50% 크기 (CSS 룰로 처리, LaTeX는 `\dfrac` 그대로).
- 그래프 라벨: foreignObject + LaTeX, 11pt + 신명조 명시.
- 마플 자료는 변형/재작성 (그대로 사용 X).
- q1은 `choices five-rows` (1열 5줄)로 고정.

### 가능한 후속 수정 후보
- 페이지 분량 한 번 더 검토 (개념 6쪽 / 문제 7쪽 / 빠정 1쪽 / 해설 2쪽 ≈ 16쪽)
- 해설 단락 분리 (다단 column break 자연스러운지)
- SVG 두 그래프의 figcaption 톤 조정
- 다른 문제 번호도 보기 길면 `five-rows` 클래스 추가

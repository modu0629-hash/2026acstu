등차수열

1) 등차수열의 정의

첫째항부터 차례로 일정한 수를 더하여 만든 수열을 등차수열이라 한다. 이때 더하는 일정한 수를 공차라 한다.

예) $2,\;4,\;6,\;8,\;10,\;\cdots$ → 첫째항 $2$, 공차 $2$인 등차수열
$\;\;\;\;\;\;5,\;2,\;-1,\;-4,\;-7,\;\cdots$ → 첫째항 $5$, 공차 $-3$인 등차수열
등차수열(等差數列): 같은 차이의 수열을 영어로 arithmetic sequence라고 하며, 공차(公差)는 common difference라고 하여 흔히 보통 $d$로 나타낸다.
2) 등차수열의 관계식

공차가 $d$인 등차수열 $\{a_n\}$에서 제 $n$항에 공차 $d$를 더하면 제 $(n+1)$항이 되므로 다음이 성립한다.

$a_{n+1} = a_n + d \;\;\Longleftrightarrow\;\; a_{n+1} - a_n = d \;\;(n=1,\,2,\,3,\,\cdots)$
3) 등차수열의 일반항

첫째항이 $a$, 공차가 $d$인 등차수열의 일반항 $a_n$은

$a_n = a + (n-1)d \;\;(n=1,\,2,\,3,\,\cdots)$

등차수열의 일반항과 등차중항

1) 일반항 $a_n$의 특징 — $n$에 대한 일차식

첫째항 $a$, 공차 $d$인 등차수열의 일반항을 정리하면

$a_n = a + (n-1)d = dn + (a-d)$

$d=A$, $a-d=B$로 놓으면 $a_n = An + B$ 꼴의 $n$에 대한 일차식이 된다.

$a_n = An + B \;\;(A,\,B$는 상수$)$

이때 첫째항은 $a_1 = A+B$, 공차는 $d = a_n - a_{n-1} = (An+B) - \{A(n-1)+B\} = A$이다.

$d=0$이면 $a_n$은 상수수열이고, $d\neq 0$이면 $a_n$은 $n$에 대한 일차식이다. 직선 $y = dx + (a-d)$ 위의 점 $(1,\,a_1),\,(2,\,a_2),\,(3,\,a_3),\,\cdots$ 이 일정한 기울기 $d$로 늘어선다고 보면 공차의 의미가 명확해진다.
2) 등차중항

세 수 $a,\,x,\,b$가 이 순서대로 등차수열을 이룰 때, $x$를 $a$와 $b$의 등차중항이라 한다. $x$가 $a$와 $b$의 등차중항이면

$b - x = x - a \;\;\Longleftrightarrow\;\; 2x = a + b \;\;\Longleftrightarrow\;\; x = \dfrac{a+b}{2}$
$x = \dfrac{a+b}{2}$ 일 때, $x$를 $a$와 $b$의 산술평균이라고도 한다.
3) 이웃하는 세 항의 등차중항

수열 $\{a_n\}$의 이웃하는 세 항 $a_{n-1},\,a_n,\,a_{n+1}$이 등차수열을 이룰 때, $a_n$은 $a_{n-1}$과 $a_{n+1}$의 등차중항이다.

$2a_n = a_{n-1} + a_{n+1} \;\;\Longleftrightarrow\;\; a_n = \dfrac{a_{n-1} + a_{n+1}}{2} \;\;(n\ge 2)$
예) 세 수 $1,\,x,\,9$가 이 순서대로 등차수열을 이루면 $x$는 $1$과 $9$의 등차중항이므로 $x = \dfrac{1+9}{2} = 5$

등차수열의 합

1) 수열의 합 $S_n$

수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합을 기호 $S_n$으로 나타낸다.

$S_n = a_1 + a_2 + a_3 + \cdots + a_n$
2) 등차수열의 합 — 가우스의 원리

첫째항이 $a$, 공차가 $d$, 끝항이 $l$인 등차수열의 합 $S_n$을 구할 때, 같은 합을 거꾸로 더해 항끼리 짝지으면 모든 짝의 합이 $a+l$로 일정해진다.

$\begin{aligned} S_n &= a + (a+d) + (a+2d) + \cdots + l\\ S_n &= l + (l-d) + (l-2d) + \cdots + a\\ \hline 2S_n &= \underbrace{(a+l)+(a+l)+\cdots+(a+l)}_{n \text{개}} = n(a+l) \end{aligned}$
3) 등차수열의 합 공식
① 첫째항 $a$, 끝항 $l$이 주어진 경우   $S_n = \dfrac{n(a+l)}{2}$
② 첫째항 $a$, 공차 $d$가 주어진 경우   $S_n = \dfrac{n\{2a + (n-1)d\}}{2}$

(②는 ①에 $l = a + (n-1)d$를 대입한 것이다.)

4) 합 $S_n$은 $n$에 대한 이차식

②를 전개하면

$S_n = \dfrac{d}{2}n^2 + \left(a - \dfrac{d}{2}\right)n$

꼴이므로 $d \neq 0$이면 $S_n$은 상수항 없는 이차식. $\dfrac{d}{2} > 0$이면 아래로 볼록(최솟값 존재), $\dfrac{d}{2} < 0$이면 위로 볼록(최댓값 존재).

$d = 0$이면 $S_n = an$ 꼴의 일차식. 상수항 없는 이차식이라는 점이 일반 수열의 합과 등차수열의 합을 가르는 결정적 차이.

수열의 합 $S_n$과 일반항 $a_n$ 사이의 관계

1) 기본 관계

수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합을 $S_n$이라 하면

$a_1 = S_1$
$a_n = S_n - S_{n-1} \;\;(n \ge 2)$
이 관계는 등차수열뿐만 아니라 모든 수열에 적용된다. $S_n - S_{n-1}$은 합의 정의에서 $a_1+\cdots+a_n$과 $a_1+\cdots+a_{n-1}$의 차로 항상 $a_n$이 되기 때문이다.
2) $S_0 = 0$의 약속

$S_0 = 0$으로 약속하면 $a_n = S_n - S_{n-1}$이 $n=1$인 경우에도 성립한다 ($a_1 = S_1 - S_0 = S_1$). 즉 첫째항부터 일반항 공식을 그대로 적용해도 되는지 여부가 자동으로 갈린다.

3) $S_n$이 $n$에 관한 이차식일 때

$S_n = An^2 + Bn + C$ ($A,\,B,\,C$는 상수, $A\neq 0$) 꼴이라 하자. $a_n = S_n - S_{n-1}$로 일반항을 구하면

$a_n = 2An + (B-A) \;\;(n \ge 2)$

$n=1$일 때 $a_1 = S_1 = A + B + C$이다. 이때 두 가지 경우로 갈린다.

① 상수항 $C = 0$일 때   $a_1$이 일반항 공식에 그대로 들어간다. $\{a_n\}$은 첫째항부터 등차수열이다.
② 상수항 $C \neq 0$일 때   $a_1$만 따로, $n \ge 2$일 때 $a_n = 2An + (B-A)$. $\{a_n\}$은 둘째항부터 등차수열이다.
"등차수열의 합 $S_n$은 상수항 없는 이차식"이라는 사실(앞 페이지)과 짝지어 외워두면, 합으로 주어진 문제에서 첫째항을 따로 다뤄야 하는지 한눈에 판정할 수 있다.

정답 및 해설

1.
[정답] (1) ④ $41$   (2) ② $31$

등차중항으로 묶기: 등차수열 $\{a_n\}$에서 $a_{k-1} + a_{k+1} = 2a_k$. 또한 $a_2 + a_3 + a_4 = 3a_3$처럼 가운데 항을 중심으로 대칭인 항들의 합은 가운데 항의 정수배가 된다.

(1) $a_1 + a_3 = 2a_2 = 2 \Rightarrow a_2 = 1$. $a_5 + a_7 = 2a_6 = 42 \Rightarrow a_6 = 21$. 두 항의 차 $a_6 - a_2 = 4d = 20 \Rightarrow d = 5$.

$\therefore a_{10} = a_6 + 4d = 21 + 20 = 41$. ④번.

(2) $a_9 = a_3 + 18 \Rightarrow 6d = 18 \Rightarrow d = 3$. $a_2 + a_3 + a_4 = 3a_3 = 21 \Rightarrow a_3 = 7$.

$\therefore a_{11} = a_3 + 8d = 7 + 24 = 31$. ②번.

2.
[정답] 공차 $3$, $\,x_5 = 17$

$2$를 첫째항, $35$를 끝항으로 두면 사이의 $10$개를 합쳐 항 수가 총 $12$. 등차수열의 일반항에서 $35 = 2 + 11d \Rightarrow d = 3$.

기울기 관점: 점 $(1,\,2)$와 점 $(12,\,35)$를 잇는 직선의 기울기가 공차이므로 $d = \dfrac{35 - 2}{12 - 1} = 3$.

$x_5$는 수열의 제 $6$항이므로 $x_5 = 2 + 5d = 17$.

3.
[정답] (1) $2,\,4,\,6$   (2) $k = -24$

(1) 세 수를 $a-d,\,a,\,a+d$로 놓으면 (등차중항 활용 — 합·곱이 $d^2$만 남음) 합 $3a = 12 \Rightarrow a = 4$. 곱 $a(a^2 - d^2) = 4(16 - d^2) = 48 \Rightarrow d^2 = 4 \Rightarrow d = \pm 2$. 세 수는 $2,\,4,\,6$.

(2) 세 실근을 $\alpha-\beta,\,\alpha,\,\alpha+\beta$로 놓고 근과 계수의 관계 적용. 합 $3\alpha = 6 \Rightarrow \alpha = 2$. 두 근씩 곱의 합 $3\alpha^2 - \beta^2 = -4 \Rightarrow 12 - \beta^2 = -4 \Rightarrow \beta^2 = 16 \Rightarrow \beta = \pm 4$.

세 근은 $-2,\,2,\,6$. 곱 $\alpha(\alpha^2 - \beta^2) = 2(4 - 16) = -24$. 근과 계수의 관계에서 세 근의 곱 $= k$이므로 $k = -24$.

검산: $x^3 - 6x^2 - 4x + 24 = (x-2)(x^2 - 4x - 12)$ $= (x-2)(x-6)(x+2)$ → 근 $-2,\,2,\,6$ 확인.

4.
[정답] $a + b = 23$

나머지정리: $f(x)$를 $x-c$로 나눈 나머지는 $f(c)$. $f(-1),\,f(1),\,f(2)$가 이 순서로 등차수열이며 $f(-2) = 0$.

$f(-1) = -1 - a + b,\;\; f(1) = 1 + a + b,\;\; f(2) = 8 + 2a + b$.

등차중항 조건: $2f(1) = f(-1) + f(2)$.   $2(1+a+b) = (-1-a+b) + (8+2a+b)$   $\Rightarrow$   $2 + 2a + 2b = 7 + a + 2b$   $\Rightarrow$   $a = 5$.

$f(-2) = -8 - 2a + b = 0 \Rightarrow b = 8 + 2a = 18$.

$\therefore a + b = 5 + 18 = 23$.

5.
[정답] $a_2 = 9$

$|a_n|$의 의미: 공차 $d \neq 0$인 등차수열에서 $|a_p| = |a_q|$ ($p \neq q$)이면 두 항의 값이 다르므로 $a_p = -a_q$, 즉 부호가 반대이다. 두 항의 등차중항(가운데 항)이 $0$이라는 뜻.

$a_3 + a_5 = 6 \Rightarrow 2a_4 = 6 \Rightarrow a_4 = 3$.

$|a_4| = |a_6|$이고 $d \neq 0$이므로 $a_4 = -a_6 \Rightarrow a_6 = -3$. (또는 $a_4$와 $a_6$의 등차중항 $a_5 = 0$.)

공차 $d = \dfrac{a_6 - a_4}{2} = \dfrac{-3 - 3}{2} = -3$.

$\therefore a_2 = a_4 - 2d = 3 - 2(-3) = 9$.

6.
[정답] $S_{30} = 90$

일정 구간으로 끊은 등차수열의 합도 등차수열을 이룬다. 즉 $S_{10},\;S_{20} - S_{10},\;S_{30} - S_{20}$이 공차가 같은 등차수열.

$S_{10} = 10,\;S_{20} - S_{10} = 30$이므로 이 등차수열의 공차는 $20$. 따라서 $S_{30} - S_{20} = 30 + 20 = 50$.

$\therefore S_{30} = S_{20} + 50 = 40 + 50 = 90$.

7.
[정답] (1) $1683$   (2) $679$

나머지로 자연수를 분할하여 등차수열로 모으고 합 공식을 적용.

(1) $100$ 이하 $3$의 배수: $3,\,6,\,9,\,\cdots,\,99$. 첫째항 $3$, 공차 $3$, 끝항 $99$. 항수 $\dfrac{99-3}{3} + 1 = 33$.

$S = \dfrac{33(3+99)}{2} = \dfrac{33 \times 102}{2} = 33 \times 51 = 1683$.

(2) $100$ 이하 자연수 중 $7$로 나누어 $3$이 남는 수: $3,\,10,\,17,\,\cdots,\,94$. ($94 = 7\times 13 + 3$이고 다음 $101 = 7\times 14 + 3$은 $100$ 초과로 제외.)

첫째항 $3$, 공차 $7$, 끝항 $94$. 항수 $\dfrac{94-3}{7} + 1 = 14$.

$S = \dfrac{14(3+94)}{2} = 7 \times 97 = 679$.

두 곡선의 차 $(x^2 + ax + b) - x^2 = ax + b$는 일차식. 따라서 $x$ 좌표가 등간격으로 늘어선 점에서 두 곡선 사이 세로 길이 $l_k$도 등차수열을 이룬다.

가우스 원리: 첫째항 $l_1 = 5$, 끝항 $l_{10} = 17$, 항수 $10$이므로

$l_1 + l_2 + \cdots + l_{10} = \dfrac{10(5 + 17)}{2} = 110$.

9.
[정답] (1) ④ $47$   (2) ③ $212$

$S_n$이 $n$에 관한 이차식이고 상수항이 있을 때: $S_n = An^2 + Bn + C$ ($C \neq 0$)이면 $n \ge 2$에서 $a_n = 2An + (B-A)$로 등차수열이지만, $a_1 = S_1$은 이 공식과 일치하지 않는다 ($S_0 \neq 0$). 즉 $a_1$은 따로, $\{a_n\}$은 둘째항부터 등차.

(1) $S_n = n^2 + 6n + 2$. 상수항 $2 \neq 0$. $a_1 = S_1 = 9$ (특수). $n \ge 2$에서 $a_n = S_n - S_{n-1} = (n^2+6n+2) - \{(n-1)^2 + 6(n-1) + 2\}$ $= 2n + 5$.

$a_5 = 15,\;a_9 = 23$. $\therefore a_1 + a_5 + a_9 = 9 + 15 + 23 = 47$. ④번.

(2) $S_n = n^2 - 2n + 3$. 상수항 $3 \neq 0$. $a_1 = S_1 = 1 - 2 + 3 = 2$ (특수). $n \ge 2$에서 $a_n = 2n - 3$.

$a_1 + a_3 + a_5 + \cdots + a_{21}$은 총 $11$개 항. $a_1 = 2$만 특수항이고, $n = 3,\,5,\,\cdots,\,21$인 $10$개 항은 $a_n = 2n - 3$을 따른다.

$a_3 = 3,\,a_5 = 7,\,\cdots,\,a_{21} = 39$ — 공차 $4$. 합 $= \dfrac{10(3+39)}{2} = 210$.

$\therefore$ 총합 $= 2 + 210 = 212$. ③번.

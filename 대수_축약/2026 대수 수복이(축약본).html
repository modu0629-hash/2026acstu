$$\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$$
변형 : ⓐ $a = 2R\sin A,\; b = 2R\sin B,\; c = 2R\sin C$     ⓑ $\sin A = \dfrac{a}{2R},\;\sin B = \dfrac{b}{2R},\;\sin C = \dfrac{c}{2R}$     ⓒ $a:b:c = \sin A:\sin B:\sin C$ (변의 비 = 사인 비)

1) 제이코사인법칙

삼각형 $\mathrm{ABC}$의 세 변의 길이 $a,\,b,\,c$와 세 각 $A,\,B,\,C$ 사이에는 다음 관계가 성립한다.

$$a^2 = b^2 + c^2 - 2bc\cos A$$ $$b^2 = c^2 + a^2 - 2ca\cos B$$ $$c^2 = a^2 + b^2 - 2ab\cos C$$

(증명) $\mathrm{A}$에서 $\mathrm{BC}$에 내린 수선의 발을 $\mathrm{H}$라 하면 직각삼각형 $\mathrm{ABH}$에서 $\overline{\mathrm{BH}} = c\cos B,\ \overline{\mathrm{AH}} = c\sin B$이므로 $\overline{\mathrm{CH}} = a - c\cos B$. 직각삼각형 $\mathrm{AHC}$에 피타고라스 정리를 적용하면 $b^2 = (c\sin B)^2 + (a - c\cos B)^2 = c^2\sin^2 B + a^2 - 2ac\cos B + c^2\cos^2 B = a^2 + c^2 - 2ac\cos B,$  즉 $b^2 = c^2 + a^2 - 2ca\cos B.$ 같은 방법으로 다른 두 식도 성립.

$\bullet$ 각 변환 : $\sin A = \dfrac{a}{2R},\ \sin B = \dfrac{b}{2R},\ \sin C = \dfrac{c}{2R}$

① $a = b$  ⇒  $\overline{\mathrm{CA}} = \overline{\mathrm{CB}}$인 이등변삼각형

② $a^2 = b^2 + c^2$  ⇒  $A = 90^\circ$인 직각삼각형

③ $a = b = c$  ⇒  정삼각형

입체도형의 최단 거리

1) 전개도를 이용한 최단 경로

곡면 위에서 두 점 사이의 최단 거리는 입체도형의 전개도로 펴서 두 점을 잇는 선분의 길이로 구한다. 전개도를 펼친 평면에서는 직선이 가장 짧기 때문이다.

2) 원뿔의 옆면 전개도

밑면의 반지름이 $r$, 모선의 길이가 $\ell$인 원뿔의 옆면을 펼치면 반지름이 $\ell$, 호의 길이가 $2\pi r$인 부채꼴이 된다. 그러므로 부채꼴의 중심각의 크기 $\alpha$는

$$\alpha = \dfrac{2\pi r}{\ell} \;(\text{rad}) = \dfrac{r}{\ell}\times 360^\circ$$
3) 최단 거리 계산 방법

[1단계] 입체도형을 전개도로 펼치고 두 점의 위치를 정확히 표시한다.

[2단계] 부채꼴의 중심각 $\alpha$를 구한다.

[3단계] 두 점을 잇는 선분에 코사인법칙을 적용해 길이를 구한다.

예) 밑면 반지름 $r$, 모선 $\ell$인 원뿔의 옆면을 펼치면 반지름 $\ell$인 부채꼴. 꼭짓점 $\mathrm{O}$에서 시작해 모선을 한 바퀴 돌아 다시 같은 모선 위 점 $\mathrm{P}$에 이르는 최단 실의 길이는 펼친 부채꼴에서 $\mathrm{P}$의 두 위치 사이의 직선거리.

$$S = \sqrt{s(s-a)(s-b)(s-c)} \quad \left(s = \dfrac{a+b+c}{2}\right)$$
헤론 공식의 한계 : 변의 길이가 무리수이거나 매우 큰 경우 $s(s-a)(s-b)(s-c)$의 계산이 매우 복잡해진다. 수치가 정수·작은 정수일 때만 실용적이며, 보통은 코사인법칙으로 한 각의 $\cos$ 값을 먼저 구해 $\sin$을 얻은 뒤 $\dfrac{1}{2}ab\sin C$를 쓰는 편이 빠르다.

등차수열

1) 등차수열의 정의

첫째항부터 차례로 일정한 수를 더하여 만든 수열을 등차수열이라 한다. 이때 더하는 일정한 수를 공차라 한다.

예) $2,\;4,\;6,\;8,\;10,\;\cdots$ → 첫째항 $2$, 공차 $2$인 등차수열
$\;\;\;\;\;\;5,\;2,\;-1,\;-4,\;-7,\;\cdots$ → 첫째항 $5$, 공차 $-3$인 등차수열
등차수열(等差數列): 같은 차이의 수열을 영어로 arithmetic sequence라고 하며, 공차(公差)는 common difference라고 하여 흔히 보통 $d$로 나타낸다.
2) 등차수열의 관계식

공차가 $d$인 등차수열 $\{a_n\}$에서 제 $n$항에 공차 $d$를 더하면 제 $(n+1)$항이 되므로 다음이 성립한다.

$a_{n+1} = a_n + d \;\;\Longleftrightarrow\;\; a_{n+1} - a_n = d \;\;(n=1,\,2,\,3,\,\cdots)$
3) 등차수열의 일반항

첫째항이 $a$, 공차가 $d$인 등차수열의 일반항 $a_n$은

$a_n = a + (n-1)d \;\;(n=1,\,2,\,3,\,\cdots)$

등차수열의 일반항과 등차중항

1) 일반항 $a_n$의 특징 — $n$에 대한 일차식

첫째항 $a$, 공차 $d$인 등차수열의 일반항을 정리하면

$a_n = a + (n-1)d = dn + (a-d)$

$d=A$, $a-d=B$로 놓으면 $a_n = An + B$ 꼴의 $n$에 대한 일차식이 된다.

$a_n = An + B \;\;(A,\,B$는 상수$)$

이때 첫째항은 $a_1 = A+B$, 공차는 $d = a_n - a_{n-1} = (An+B) - \{A(n-1)+B\} = A$이다.

$d=0$이면 $a_n$은 상수수열이고, $d\neq 0$이면 $a_n$은 $n$에 대한 일차식이다. 직선 $y = dx + (a-d)$ 위의 점 $(1,\,a_1),\,(2,\,a_2),\,(3,\,a_3),\,\cdots$ 이 일정한 기울기 $d$로 늘어선다고 보면 공차의 의미가 명확해진다.
2) 등차중항

세 수 $a,\,x,\,b$가 이 순서대로 등차수열을 이룰 때, $x$를 $a$와 $b$의 등차중항이라 한다. $x$가 $a$와 $b$의 등차중항이면

$b - x = x - a \;\;\Longleftrightarrow\;\; 2x = a + b \;\;\Longleftrightarrow\;\; x = \dfrac{a+b}{2}$
$x = \dfrac{a+b}{2}$ 일 때, $x$를 $a$와 $b$의 산술평균이라고도 한다.
3) 이웃하는 세 항의 등차중항

수열 $\{a_n\}$의 이웃하는 세 항 $a_{n-1},\,a_n,\,a_{n+1}$이 등차수열을 이룰 때, $a_n$은 $a_{n-1}$과 $a_{n+1}$의 등차중항이다.

$2a_n = a_{n-1} + a_{n+1} \;\;\Longleftrightarrow\;\; a_n = \dfrac{a_{n-1} + a_{n+1}}{2} \;\;(n\ge 2)$
예) 세 수 $1,\,x,\,9$가 이 순서대로 등차수열을 이루면 $x$는 $1$과 $9$의 등차중항이므로 $x = \dfrac{1+9}{2} = 5$

등차수열의 합

1) 수열의 합 $S_n$

수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합을 기호 $S_n$으로 나타낸다.

$S_n = a_1 + a_2 + a_3 + \cdots + a_n$
2) 등차수열의 합 — 가우스의 원리

첫째항이 $a$, 공차가 $d$, 끝항이 $l$인 등차수열의 합 $S_n$을 구할 때, 같은 합을 거꾸로 더해 항끼리 짝지으면 모든 짝의 합이 $a+l$로 일정해진다.

$\begin{aligned} S_n &= a + (a+d) + (a+2d) + \cdots + l\\ S_n &= l + (l-d) + (l-2d) + \cdots + a\\ \hline 2S_n &= \underbrace{(a+l)+(a+l)+\cdots+(a+l)}_{n \text{개}} = n(a+l) \end{aligned}$
3) 등차수열의 합 공식
① 첫째항 $a$, 끝항 $l$이 주어진 경우   $S_n = \dfrac{n(a+l)}{2}$
② 첫째항 $a$, 공차 $d$가 주어진 경우   $S_n = \dfrac{n\{2a + (n-1)d\}}{2}$

(②는 ①에 $l = a + (n-1)d$를 대입한 것이다.)

4) 합 $S_n$은 $n$에 대한 이차식

②를 전개하면

$S_n = \dfrac{d}{2}n^2 + \left(a - \dfrac{d}{2}\right)n$

꼴이므로 $d \neq 0$이면 $S_n$은 상수항 없는 이차식. $\dfrac{d}{2} > 0$이면 아래로 볼록(최솟값 존재), $\dfrac{d}{2} < 0$이면 위로 볼록(최댓값 존재).

$d = 0$이면 $S_n = an$ 꼴의 일차식. 상수항 없는 이차식이라는 점이 일반 수열의 합과 등차수열의 합을 가르는 결정적 차이.

수열의 합 $S_n$과 일반항 $a_n$ 사이의 관계

1) 기본 관계

수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합을 $S_n$이라 하면

$a_1 = S_1$
$a_n = S_n - S_{n-1} \;\;(n \ge 2)$
이 관계는 등차수열뿐만 아니라 모든 수열에 적용된다. $S_n - S_{n-1}$은 합의 정의에서 $a_1+\cdots+a_n$과 $a_1+\cdots+a_{n-1}$의 차로 항상 $a_n$이 되기 때문이다.
2) $S_0 = 0$의 약속

$S_0 = 0$으로 약속하면 $a_n = S_n - S_{n-1}$이 $n=1$인 경우에도 성립한다 ($a_1 = S_1 - S_0 = S_1$). 즉 첫째항부터 일반항 공식을 그대로 적용해도 되는지 여부가 자동으로 갈린다.

3) $S_n$이 $n$에 관한 이차식일 때

$a_n = 2An + (B-A) \;\;(n \ge 2)$

$n=1$일 때 $a_1 = S_1 = A + B + C$이다. 이때 두 가지 경우로 갈린다.

① 상수항 $C = 0$일 때   $a_1$이 일반항 공식에 그대로 들어간다. $\{a_n\}$은 첫째항부터 등차수열이다.
② 상수항 $C \neq 0$일 때   $a_1$만 따로, $n \ge 2$일 때 $a_n = 2An + (B-A)$. $\{a_n\}$은 둘째항부터 등차수열이다.
"등차수열의 합 $S_n$은 상수항 없는 이차식"이라는 사실(앞 페이지)과 짝지어 외워두면, 합으로 주어진 문제에서 첫째항을 따로 다뤄야 하는지 한눈에 판정할 수 있다.

등비수열

등비수열의 정의와 일반항

1) 등비수열의 정의

첫째항부터 차례로 일정한 수를 곱하여 만든 수열을 등비수열이라 하고, 곱하는 일정한 수를 공비라 한다.

예) $1,\,-3,\,9,\,-27,\,81,\,\cdots$ : 첫째항 $1$, 공비 $-3$인 등비수열
$4,\,2,\,1,\,\dfrac{1}{2},\,\dfrac{1}{4},\,\cdots$ : 첫째항 $4$, 공비 $\dfrac{1}{2}$인 등비수열
등비수열(等比數列)은 영어로 geometric sequence, 공비(公比)는 common ratio라 한다. 첫째항과 공비가 $0$이 아닌 등비수열만 다룬다.
2) 등비수열의 일반항

첫째항이 $a$, 공비가 $r$인 등비수열 $\{a_n\}$의 일반항은

$$a_n = a\,r^{\,n-1} \qquad (n=1,\,2,\,3,\,\cdots)$$
3) 등비수열의 관계식

공비가 $r$인 등비수열 $\{a_n\}$의 이웃하는 두 항 $a_n,\,a_{n+1}$ 사이에는 다음 관계식이 성립한다.

$$\dfrac{a_{n+1}}{a_n} = r \quad\Longleftrightarrow\quad a_{n+1} = r\,a_n$$
4) 등비 중항

$0$이 아닌 세 수 $a,\,b,\,c$가 이 순서대로 등비수열을 이룰 때, $b$를 $a$와 $c$의 등비 중항이라 한다.

$\dfrac{b}{a} = \dfrac{c}{b} \;\Longleftrightarrow\; b^2 = ac$
예) 세 항이 등비수열을 이루는 문제에서 미지수를 줄이려면 ① 등비 중항을 이용하여 $b^2 = ac$를 적용하거나, ② 세 항을 $a,\,ar,\,ar^2$으로 두는 방법을 자주 쓴다.

등비수열의 활용 — 두 직선에 접하는 원의 반지름

한 점에서 만나는 두 직선에 동시에 접하면서 차례로 외접하는 원들의 반지름은 등비수열을 이룬다. 공비는 두 직선의 사잇각만으로 결정된다.

한 점 $V$에서 만나는 두 직선의 사잇각이 $2\alpha$이고, 두 직선에 동시에 접하면서 차례로 외접하는 원들의 반지름을 $r_1,\,r_2,\,r_3,\,\cdots$이라 하면 $\{r_n\}$은 공비가 $\dfrac{1+\sin\alpha}{1-\sin\alpha}$인 등비수열을 이룬다.

(증명) 두 직선의 교점을 $V$, $n$번째 원의 중심을 $C_n$이라 하면 $C_n$은 두 직선의 양분선 위에 있고, $C_n$에서 두 직선까지의 거리가 곧 $r_n$이므로 $\overline{VC_n}\sin\alpha = r_n$,  즉 $\overline{VC_n} = \dfrac{r_n}{\sin\alpha}$. 인접한 두 원이 외접하므로 $\overline{VC_{n+1}} - \overline{VC_n} = r_{n+1} + r_n$이고, 위 식을 대입하면 $\dfrac{r_{n+1}-r_n}{\sin\alpha} = r_{n+1}+r_n \;\Longleftrightarrow\; r_{n+1}(1-\sin\alpha) = r_n(1+\sin\alpha)$
$\therefore\;\dfrac{r_{n+1}}{r_n} = \dfrac{1+\sin\alpha}{1-\sin\alpha}\;(\text{일정})$

① $a_n = A\,r^{\,n}$  꼴
② $S_n = A\,r^{\,n} + B$  꼴 (단, $r \ne 1$)

역으로 합 $S_n$이 $A\,r^{\,n} + B$ 꼴로 주어진 수열에서 일반항 $a_n$을 구하면 다음을 얻는다.

ⅰ) $B = -A$  ⇒   $\{a_n\}$은 첫째항부터 등비수열을 이룬다.
ⅱ) $B \ne -A$  ⇒   $\{a_n\}$은 두 번째 항부터 등비수열을 이룬다.

$(\text{ii})\;$ $n=k$일 때 $\,p(n)$이 성립한다고 가정하면, $\,n=k+1$일 때에도 $\,p(n)$이 성립한다.
2) $n\geq m$인 모든 자연수에 대한 명제 증명

명제 $p(n)$이 $n\geq m$($m$은 자연수)인 모든 자연수 $n$에 대하여 성립함을 증명하려면 다음 두 가지를 보이면 된다.

$\;(\text{i})\;$ $n=m$일 때 $\,p(n)$이 성립한다.
$(\text{ii})\;$ $n=k\,(k\geq m)$일 때 $\,p(n)$이 성립한다고 가정하면, $\,n=k+1$일 때에도 $\,p(n)$이 성립한다.

이와 같은 방법으로 자연수에 대한 어떤 명제가 참임을 증명하는 방법을 수학적 귀납법이라 한다.

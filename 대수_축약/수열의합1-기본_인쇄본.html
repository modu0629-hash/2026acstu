수열의 합

합의 기호 $\Sigma$의 정의

1) $\Sigma$의 정의

수열 $\{a_n\}$에서 첫째항부터 제 $n$항까지의 합 $a_1 + a_2 + a_3 + \cdots + a_n$을 합의 기호 $\Sigma$를 사용하여 다음과 같이 나타낸다.

$\displaystyle\sum_{k=1}^{n} a_k = a_1 + a_2 + a_3 + \cdots + a_n$
$\Sigma$는 합(summation)을 뜻하는 그리스 문자 시그마(sigma)의 첫 글자이다. $k$ 대신 $i,\,j,\,l$ 등 다른 문자를 써도 된다.
2) $\Sigma$의 뜻

$\displaystyle\sum_{k=1}^{n} a_k$는 일반항 $a_k$의 $k$에 $1,\,2,\,3,\,\cdots,\,n$을 차례로 대입하여 더한다는 뜻이다.

예) $\displaystyle\sum_{k=1}^{5} k^2 = 1^2+2^2+3^2+4^2+5^2 = 55$
$\displaystyle\sum_{k=1}^{4}(2k-1) = 1+3+5+7 = 16$,   $\displaystyle\sum_{k=3}^{6} k = 3+4+5+6 = 18$

$\Sigma$의 기본 성질

1) 합과 차의 성질
$\displaystyle\sum_{k=1}^{n}(a_k + b_k) = \sum_{k=1}^{n} a_k + \sum_{k=1}^{n} b_k,\;\;\; \displaystyle\sum_{k=1}^{n}(a_k - b_k) = \sum_{k=1}^{n} a_k - \sum_{k=1}^{n} b_k$
2) 상수배의 성질
$\displaystyle\sum_{k=1}^{n} c\,a_k = c\sum_{k=1}^{n} a_k$  ($c$는 상수)
3) 상수의 성질
$\displaystyle\sum_{k=1}^{n} c = nc$  ($c$는 상수)
$\Sigma$는 항별로 분배되지만 곱과 거듭제곱에는 분배되지 않는다. $\displaystyle\sum a_k\,b_k \ne \sum a_k\cdot\sum b_k$, $\displaystyle\sum a_k^{\,2} \ne \left(\sum a_k\right)^{\!2}.$
4) 합 $S_n$과 일반항 $a_n$의 관계

수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합을 $S_n$이라 하면

$a_n = \begin{cases} S_1 & (n=1) \\ S_n - S_{n-1} & (n \ge 2) \end{cases}$

자연수의 거듭제곱의 합

1) 자연수·자연수의 제곱·자연수의 세제곱의 합
(1) $\displaystyle\sum_{k=1}^{n} k = 1+2+3+\cdots+n = \dfrac{n(n+1)}{2}$
(2) $\displaystyle\sum_{k=1}^{n} k^2 = 1^2+2^2+3^2+\cdots+n^2 = \dfrac{n(n+1)(2n+1)}{6}$
(3) $\displaystyle\sum_{k=1}^{n} k^3 = 1^3+2^3+3^3+\cdots+n^3 = \left\{\dfrac{n(n+1)}{2}\right\}^{\!2}$
2) 연속한 자연수의 곱의 합
(1) $\displaystyle\sum_{k=1}^{n} k(k+1) = \dfrac{n(n+1)(n+2)}{3}$
(2) $\displaystyle\sum_{k=1}^{n} k(k+1)(k+2) = \dfrac{n(n+1)(n+2)(n+3)}{4}$
일반적으로 $\displaystyle\sum_{k=1}^{n} k(k+1)(k+2)\cdots(k+m) = \dfrac{n(n+1)(n+2)\cdots(n+m+1)}{m+2}$. $\Sigma$를 직접 전개하지 않고 묶음으로 빠르게 계산할 때 자주 쓴다.

특수한 수열의 합

1) 지수꼴 — 등비수열의 합

일반항이 $a r^{\,k-1}$ 꼴인 등비수열의 합은 등비수열의 합 공식으로 정리한다.

$\displaystyle\sum_{k=1}^{n} a\,r^{\,k-1} = \dfrac{a(r^{\,n}-1)}{r-1}$  ($r\ne 1$)
예) $\displaystyle\sum_{k=m}^{n} r^{\,k} = r^{\,m} + r^{\,m+1} + \cdots + r^{\,n} = \dfrac{r^{\,m}(r^{\,n-m+1}-1)}{r-1}$  (공비 $r$, 항의 개수 $n-m+1$)
$\displaystyle\sum_{k=1}^{n} r^{\,3k-1} = r^2 + r^5 + \cdots + r^{\,3n-1} = \dfrac{r^2\{(r^3)^{\,n}-1\}}{r^3-1}$  (공비 $r^3$)
2) 같은 숫자가 반복된 수열의 합

한 자리 수 $a$가 한 자리, 두 자리, 세 자리, $\ldots$ 로 반복되는 수의 합 ($a$가 $n$개)

$a + \underbrace{aa}_{2자리} + \underbrace{aaa}_{3자리} + \cdots + \underbrace{aa\cdots a}_{n자리}$ $\;=\; \dfrac{a}{9}\bigl\{(10-1)+(10^2-1)+\cdots+(10^{\,n}-1)\bigr\}$ $\;=\; \dfrac{a(10^{\,n+1} - 9n - 10)}{81}$
$\displaystyle\underbrace{99\cdots 9}_{n자리} = 10^{\,n} - 1$임을 이용해 $9$로 묶고 $\Sigma$ 공식을 적용한다.

정답 및 해설

1.
[정답] (1) $120$   (2) $3280$   (3) $107$

(1) $\Sigma$ 분배 + 자연수의 합 공식. $\displaystyle\sum_{k=1}^{10}(2k+1) = 2\sum k + \sum 1 = 2\cdot\dfrac{10\cdot 11}{2} + 10 = 120.$

(2) 등비합. 첫째항 $1$, 공비 $3$, 항 $8$개. $\displaystyle\sum_{k=1}^{8} 3^{\,k-1} = \dfrac{3^8-1}{3-1} = \dfrac{6560}{2} = 3280.$

(3) 분배. $\displaystyle\sum 2^{\,k} = \dfrac{2(2^5-1)}{1} = 62,\;3\sum k = 3\cdot 15 = 45.$ 합 $107.$

2.
[정답] (1) $240$   (2) $20$

(1) 상수배의 성질. $\displaystyle\sum 3(a_{k+1}-a_k) = 3\sum (a_{k+1}-a_k) = 3\cdot 80 = 240.$

(2) 망원합(telescoping). $\displaystyle\sum_{k=1}^{20}(a_{k+1}-a_k) = a_{21} - a_1 = 30 - 10 = 20.$

3.
[정답] $1540$

$S_n = n^2 + n.$ $n\ge 2$일 때 $a_n = S_n - S_{n-1} = (n^2+n) - ((n-1)^2 + (n-1)) = 2n.$

$n=1$일 때 $a_1 = S_1 = 2 = 2\cdot 1$ — 일치. 따라서 $a_n = 2n.$

$\displaystyle\sum_{k=1}^{10} a_k^{\,2} = \sum (2k)^2 = 4\sum k^2 = 4\cdot\dfrac{10\cdot 11\cdot 21}{6} = 4\cdot 385 = 1540.$

4.
[정답] (1) $\dfrac{4n(n+1)(n+2)}{3}+n$   (2) $605$

(1) $(2k+1)^2 = 4k(k+1) + 1$로 변형하면 연속 자연수 곱의 합 공식 적용 편하다.

$\displaystyle\sum_{k=1}^{n}(2k+1)^2 = 4\sum k(k+1) + \sum 1 = 4\cdot\dfrac{n(n+1)(n+2)}{3} + n.$

(2) 일반항 $k\,(k+4)$ 꼴. $\displaystyle\sum_{k=1}^{10} k(k+4) = \sum k^2 + 4\sum k = 385 + 4\cdot 55 = 385 + 220 = 605.$

5.
[정답] (1) $1540$   (2) $\dfrac{n(n+1)(n+2)}{6}$

(1) 일반항 $k(21-k).$ $\displaystyle\sum_{k=1}^{20} k(21-k) = 21\sum k - \sum k^2 = 21\cdot 210 - 2870 = 1540.$

(2) 일반항 $k(n+1-k).$ $\displaystyle\sum_{k=1}^{n} k(n+1-k) = (n+1)\sum k - \sum k^2$ $= \dfrac{n(n+1)^2}{2} - \dfrac{n(n+1)(2n+1)}{6}$ $= \dfrac{n(n+1)}{6}\bigl[3(n+1)-(2n+1)\bigr]$ $= \dfrac{n(n+1)(n+2)}{6}.$

6.
[정답] $4960$

$k$층의 블록 수 $T_k = 1+2+\cdots+k = \dfrac{k(k+1)}{2}$ (삼각수). $1,\,3,\,6,\,10,\,\ldots$ 와 일치.

$30$층 총 블록 수 $= \displaystyle\sum_{k=1}^{30}\dfrac{k(k+1)}{2} = \dfrac{1}{2}\sum k(k+1)$ $= \dfrac{1}{2}\cdot\dfrac{30\cdot 31\cdot 32}{3} = \dfrac{30\cdot 31\cdot 32}{6} = 4960.$

7.
[정답] (1) $\dfrac{n(n+1)(n+2)}{6}$   (2) $2^{\,n+1}-n-2$

(1) 일반항 $a_k = 1+2+\cdots+k = \dfrac{k(k+1)}{2}.$ 합 $S_n = \displaystyle\sum_{k=1}^{n}\dfrac{k(k+1)}{2} = \dfrac{n(n+1)(n+2)}{6}.$

(2) 일반항 $a_k = 1+2+4+\cdots+2^{\,k-1} = 2^{\,k}-1$ (등비합).

$S_n = \displaystyle\sum_{k=1}^{n}(2^{\,k}-1) = \dfrac{2(2^{\,n}-1)}{1} - n = 2^{\,n+1} - n - 2.$

8.
[정답] (1) $35$   (2) $\dfrac{n(n+1)(n+5)}{6}$

(1) 안쪽 합 $\displaystyle\sum_{j=1}^{k} j = \dfrac{k(k+1)}{2}.$

$\displaystyle\sum_{k=1}^{5}\dfrac{k(k+1)}{2} = \dfrac{1}{2}\cdot\dfrac{5\cdot 6\cdot 7}{3} = \dfrac{210}{6} = 35.$

(2) 안쪽 합 $\displaystyle\sum_{j=1}^{k}(j+1) = \dfrac{k(k+1)}{2} + k = \dfrac{k(k+3)}{2}.$

$\displaystyle\sum_{k=1}^{n}\dfrac{k(k+3)}{2} = \dfrac{1}{2}\!\left[\dfrac{n(n+1)(2n+1)}{6} + 3\cdot\dfrac{n(n+1)}{2}\right]$ $= \dfrac{n(n+1)}{12}\bigl[(2n+1)+9\bigr] = \dfrac{n(n+1)(n+5)}{6}.$

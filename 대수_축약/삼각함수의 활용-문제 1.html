정답 및 해설

1.
[정답] ① $10$

$\angle\mathrm{DAB} = \theta$라 하면 삼각형 $\mathrm{ABD}$에서 코사인법칙에 의하여

$\cos\theta = \dfrac{4^2 + 5^2 - (\sqrt{33})^2}{2\cdot 4\cdot 5} = \dfrac{8}{40} = \dfrac{1}{5}.$

$\sin\theta = \sqrt{1 - \dfrac{1}{25}} = \dfrac{2\sqrt{6}}{5}.$

사각형 $\mathrm{ABCD}$가 한 원에 내접하므로 $\angle\mathrm{BCD} = \pi - \theta.$

따라서 $\sin(\angle\mathrm{BCD}) = \sin(\pi - \theta) = \sin\theta = \dfrac{2\sqrt{6}}{5}.$

삼각형 $\mathrm{BCD}$의 넓이가 $2\sqrt{6}$이므로

$\dfrac{1}{2}\cdot\overline{\mathrm{BC}}\cdot\overline{\mathrm{CD}}\cdot\dfrac{2\sqrt{6}}{5} = 2\sqrt{6}.$

$\therefore \overline{\mathrm{BC}}\times\overline{\mathrm{CD}} = 10.$

2.
[정답] ① $6$

외접원의 넓이가 $50\pi$이므로 반지름의 길이를 $R$라 하면 $R = 5\sqrt{2}.$

$\overline{\mathrm{AB}} = \sqrt{2}\,k,\ \overline{\mathrm{AC}} = k\;(k \gt 0)$로 놓고 $\angle\mathrm{ABC} = \alpha$라 하면 사인법칙에 의하여

$\dfrac{k}{\sin\alpha} = 2R = 10\sqrt{2},$  즉 $\sin\alpha = \dfrac{k}{10\sqrt{2}}.$  ⋯ ㉠

직각삼각형 $\mathrm{ABH}$에서 $\sin\alpha = \dfrac{2}{\sqrt{2}\,k}.$  ⋯ ㉡

㉠, ㉡을 연립하면 $\dfrac{k}{10\sqrt{2}} = \dfrac{2}{\sqrt{2}\,k},$  $k^2 = 20.$

직각삼각형 $\mathrm{ABH}$에서 피타고라스의 정리에 의하여

$\overline{\mathrm{BH}} = \sqrt{\overline{\mathrm{AB}}^{\,2} - \overline{\mathrm{AH}}^{\,2}} = \sqrt{2k^2 - 4} = \sqrt{40 - 4} = 6.$

3.
[정답] $12$

삼각형 $\mathrm{ABC}$에서 사인법칙에 의하여 $\dfrac{\overline{\mathrm{BC}}}{\sin A} = \dfrac{\overline{\mathrm{AC}}}{\sin B}$이고 $2\sin A = \sin B$이므로 $\overline{\mathrm{AC}} = 2\overline{\mathrm{BC}}.$

$\overline{\mathrm{BC}} = k$라 하면 $\overline{\mathrm{AC}} = 2k.$

삼각형 $\mathrm{ABC}$에서 코사인법칙에 의하여

$6^2 = k^2 + (2k)^2 - 2\cdot k\cdot 2k\cdot\dfrac{4}{5}.$

$\dfrac{9}{5}k^2 = 36,$ $k \gt 0$이므로 $k = 2\sqrt{5}.$

$\cos C = \dfrac{4}{5}$에서 $\sin C = \sqrt{1 - \left(\dfrac{4}{5}\right)^2} = \dfrac{3}{5}.$

따라서 삼각형 $\mathrm{ABC}$의 넓이는

$\dfrac{1}{2}\times 2\sqrt{5}\times 4\sqrt{5}\times\dfrac{3}{5} = 12.$

4.
[정답] ⑤ $\dfrac{64\sqrt{2}}{9}$

외접원의 넓이가 $9\pi$이므로 반지름의 길이를 $R$라 하면 $R = 3.$

조건 (가)에서 $3\sin A = 2\sin B$이므로 사인법칙에 의해 $\dfrac{3a}{2R} = \dfrac{2b}{2R},$  $\therefore 3a = 2b.$

조건 (나)에서 $\cos B = \cos C$이므로 $\angle\mathrm{B} = \angle\mathrm{C},$  $\therefore b = c.$

코사인법칙에 의하여

$\cos A = \dfrac{b^2 + c^2 - a^2}{2bc} = \dfrac{b^2 + b^2 - \left(\frac{2}{3}b\right)^2}{2b^2} = \dfrac{7}{9}.$

$\sin A = \sqrt{1 - \left(\dfrac{7}{9}\right)^2} = \dfrac{4\sqrt{2}}{9}.$

사인법칙 $\dfrac{a}{\sin A} = 2R$에서 $a = 2\times 3\times\dfrac{4\sqrt{2}}{9} = \dfrac{8\sqrt{2}}{3}.$

따라서 삼각형 $\mathrm{ABC}$의 넓이를 $S$라 하면

$S = \dfrac{1}{2}bc\sin A = \dfrac{1}{2}\cdot\dfrac{3}{2}a\cdot\dfrac{3}{2}a\cdot\dfrac{4\sqrt{2}}{9} = \dfrac{\sqrt{2}}{2}a^2.$

$S = \dfrac{\sqrt{2}}{2}\left(\dfrac{8\sqrt{2}}{3}\right)^2 = \dfrac{64\sqrt{2}}{9}.$

5.
[정답] $26$

$\dfrac{\sin\beta}{\sin\alpha} = \dfrac{3}{2}$이므로 사인법칙에 의하여 두 원의 반지름의 길이의 비는 $3 : 2.$

(공통변 $\overline{\mathrm{AC}}$에 대해 $\overline{\mathrm{AC}} = 2R\sin\alpha = 2R'\sin\beta$이므로 $R : R' = \sin\beta : \sin\alpha = 3 : 2.$)

각각의 반지름의 길이를 $3r,\ 2r$라 하면 삼각형 $\mathrm{AOO'}$에서 $\overline{\mathrm{AO}} = 3r,\ \overline{\mathrm{AO'}} = 2r,\ \angle\mathrm{OAO'} = \pi - (\alpha + \beta).$

코사인법칙에 의하여 $1 = 9r^2 + 4r^2 - 12r^2\cos(\pi - (\alpha+\beta)) = 13r^2 + 4r^2 = 17r^2.$

($\cos(\pi - (\alpha+\beta)) = -\cos(\alpha+\beta) = -\dfrac{1}{3}$ 대입.)

$\therefore r^2 = \dfrac{1}{17}.$

삼각형 $\mathrm{ABC}$의 외접원의 넓이 $S = (3r)^2\pi = 9r^2\pi = \dfrac{9}{17}\pi.$

$\therefore p = 17,\ q = 9,\ p + q = 26.$

6.
[정답] $21$

외접원의 반지름의 길이가 $7$이므로 사인법칙을 이용하면

$\dfrac{\overline{\mathrm{BC}}}{\sin\frac{\pi}{3}} = 2R = 14.$

$\therefore \overline{\mathrm{BC}} = 14\sin\dfrac{\pi}{3} = 7\sqrt{3}.$

$\overline{\mathrm{AB}} : \overline{\mathrm{AC}} = 3 : 1$이므로 $\overline{\mathrm{AB}} = 3k,\ \overline{\mathrm{AC}} = k$로 두자.

코사인법칙에 의하여

$\overline{\mathrm{BC}}^{\,2} = \overline{\mathrm{AB}}^{\,2} + \overline{\mathrm{AC}}^{\,2} - 2\cdot\overline{\mathrm{AB}}\cdot\overline{\mathrm{AC}}\cos\dfrac{\pi}{3}.$

$(7\sqrt{3})^2 = (3k)^2 + k^2 - 2\cdot 3k\cdot k\cdot\dfrac{1}{2} = 9k^2 + k^2 - 3k^2 = 7k^2.$

$147 = 7k^2,$  $\therefore k^2 = 21.$

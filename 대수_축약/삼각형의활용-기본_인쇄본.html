삼각형의 활용

기본 상식

1) 삼각형의 기본 성질

① 삼각형에서 두 변의 길이의 합은 다른 한 변의 길이보다 크다.

② 세 내각의 크기의 합은 $180^\circ$이다.

③ 삼각형의 6요소 — 세 변의 길이 $a,\,b,\,c$와 세 내각 $A,\,B,\,C$.

2) 사인법칙

외접원의 반지름의 길이가 $R$인 삼각형 $\mathrm{ABC}$에서 세 변과 세 각의 사인 사이에는 다음 관계가 성립한다.

$$\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$$
변형 : ⓐ $a = 2R\sin A,\; b = 2R\sin B,\; c = 2R\sin C$     ⓑ $\sin A = \dfrac{a}{2R},\;\sin B = \dfrac{b}{2R},\;\sin C = \dfrac{c}{2R}$     ⓒ $a:b:c = \sin A:\sin B:\sin C$ (변의 비 = 사인 비)
3) 사인법칙을 사용하는 경우

① 한 변과 두 각의 크기를 알 때 — 나머지 두 변

② 두 변과 끼인각이 아닌 한 각의 크기를 알 때 — 나머지 각·변

③ 외접원의 반지름이 주어졌거나 구해야 할 때

코사인 법칙

1) 제이코사인법칙

삼각형 $\mathrm{ABC}$의 세 변의 길이 $a,\,b,\,c$와 세 각 $A,\,B,\,C$ 사이에는 다음 관계가 성립한다.

$$a^2 = b^2 + c^2 - 2bc\cos A$$ $$b^2 = c^2 + a^2 - 2ca\cos B$$ $$c^2 = a^2 + b^2 - 2ab\cos C$$

(증명) $\mathrm{A}$에서 $\mathrm{BC}$에 내린 수선의 발을 $\mathrm{H}$라 하면 직각삼각형 $\mathrm{ABH}$에서 $\overline{\mathrm{BH}} = c\cos B,\ \overline{\mathrm{AH}} = c\sin B$이므로 $\overline{\mathrm{CH}} = a - c\cos B$. 직각삼각형 $\mathrm{AHC}$에 피타고라스 정리를 적용하면 $b^2 = (c\sin B)^2 + (a - c\cos B)^2 = c^2\sin^2 B + a^2 - 2ac\cos B + c^2\cos^2 B = a^2 + c^2 - 2ac\cos B,$  즉 $b^2 = c^2 + a^2 - 2ca\cos B.$ 같은 방법으로 다른 두 식도 성립.

2) 각을 변으로 나타내기 (코사인의 변형)
$$\cos A = \dfrac{b^2 + c^2 - a^2}{2bc},\quad \cos B = \dfrac{c^2 + a^2 - b^2}{2ca},\quad \cos C = \dfrac{a^2 + b^2 - c^2}{2ab}$$

삼각형의 모양 결정

각이 포함된 식이 주어지면 사인법칙·코사인법칙을 이용하여 모든 각을 변으로 표현한 다음 변의 관계를 확인한다.

$\bullet$ 각 변환 : $\sin A = \dfrac{a}{2R},\ \sin B = \dfrac{b}{2R},\ \sin C = \dfrac{c}{2R}$
$\bullet$ 코사인 변환 : $\cos A = \dfrac{b^2+c^2-a^2}{2bc},\ \cos B = \dfrac{c^2+a^2-b^2}{2ca},\ \cos C = \dfrac{a^2+b^2-c^2}{2ab}$
대표적인 삼각형 판정

① $a = b$  ⇒  $\overline{\mathrm{CA}} = \overline{\mathrm{CB}}$인 이등변삼각형

② $a^2 = b^2 + c^2$  ⇒  $A = 90^\circ$인 직각삼각형

③ $a = b = c$  ⇒  정삼각형

예) $a\sin A = b\sin B$를 정리하면 사인법칙으로 $a^2 = b^2$이므로 $a = b$인 이등변삼각형.
예) $\cos A = 0$이면 $A = 90^\circ$이므로 $A$가 직각인 직각삼각형.

사인법칙 · 코사인법칙의 활용

1) 내각의 이등분선 — 보각 관계

$\angle A$의 이등분선이 변 $\mathrm{BC}$와 만나는 점을 $\mathrm{D}$라 하자. 두 작은 삼각형 $\mathrm{ABD},\,\mathrm{ACD}$에서 $\angle\mathrm{ADB} + \angle\mathrm{ADC} = 180^\circ$이므로 두 각은 보각 관계이다.

$\bullet$ 보각 관계 : $\cos(\angle\mathrm{ADC}) = -\cos(\angle\mathrm{ADB})$
$\bullet$ 두 작은 삼각형에 코사인법칙을 각각 쓰고 두 식을 더하면 $\cos$ 항이 소거되어 길이 관계식이 나온다.
2) 원의 활용

① 원주각 : 한 호에 대한 원주각의 크기는 같다 → 같은 호에서 원주각이 같으면 그 호의 길이(현의 길이)도 같다. (역도 성립)

② 원에 내접하는 사각형의 대각의 합은 $180^\circ$이다.   ($A + C = B + D = 180^\circ$)

③ 외접원의 반지름이 주어지면 사인법칙 $\dfrac{a}{\sin A} = 2R$로 변·각 관계를 잡는다.

④ 내접원의 반지름이 주어지면 삼각형의 넓이 $S = \dfrac{1}{2}r(a+b+c)$를 이용한다.

원에 내접하는 사각형 $\mathrm{ABCD}$에서 $\cos B = -\cos D$이므로 두 삼각형 $\mathrm{ABC},\,\mathrm{ACD}$에 코사인법칙을 쓰면 $\cos B$의 부호 차로 변 관계를 빠르게 정리할 수 있다.

입체도형의 최단 거리

1) 전개도를 이용한 최단 경로

곡면 위에서 두 점 사이의 최단 거리는 입체도형의 전개도로 펴서 두 점을 잇는 선분의 길이로 구한다. 전개도를 펼친 평면에서는 직선이 가장 짧기 때문이다.

2) 원뿔의 옆면 전개도

밑면의 반지름이 $r$, 모선의 길이가 $\ell$인 원뿔의 옆면을 펼치면 반지름이 $\ell$, 호의 길이가 $2\pi r$인 부채꼴이 된다. 그러므로 부채꼴의 중심각의 크기 $\alpha$는

$$\alpha = \dfrac{2\pi r}{\ell} \;(\text{rad}) = \dfrac{r}{\ell}\times 360^\circ$$
3) 최단 거리 계산 방법

[1단계] 입체도형을 전개도로 펼치고 두 점의 위치를 정확히 표시한다.

[2단계] 부채꼴의 중심각 $\alpha$를 구한다.

[3단계] 두 점을 잇는 선분에 코사인법칙을 적용해 길이를 구한다.

예) 밑면 반지름 $r$, 모선 $\ell$인 원뿔의 옆면을 펼치면 반지름 $\ell$인 부채꼴. 꼭짓점 $\mathrm{O}$에서 시작해 모선을 한 바퀴 돌아 다시 같은 모선 위 점 $\mathrm{P}$에 이르는 최단 실의 길이는 펼친 부채꼴에서 $\mathrm{P}$의 두 위치 사이의 직선거리.

삼각형의 넓이

삼각형 $\mathrm{ABC}$의 넓이를 $S$라 할 때, 주어진 조건에 따라 다음을 사용한다.

1) 두 변과 끼인각의 크기를 알 때
$$S = \dfrac{1}{2}ab\sin C = \dfrac{1}{2}bc\sin A = \dfrac{1}{2}ca\sin B$$
2) 외접원의 반지름의 길이 $R$이 주어진 경우
$$S = \dfrac{abc}{4R} = 2R^2\sin A\sin B\sin C$$
3) 내접원의 반지름의 길이 $r$이 주어진 경우
$$S = \dfrac{1}{2}r(a+b+c)$$
4) 헤론의 공식 — 세 변의 길이를 알 때
$$S = \sqrt{s(s-a)(s-b)(s-c)} \quad \left(s = \dfrac{a+b+c}{2}\right)$$
헤론 공식의 한계 : 변의 길이가 무리수이거나 매우 큰 경우 $s(s-a)(s-b)(s-c)$의 계산이 매우 복잡해진다. 수치가 정수·작은 정수일 때만 실용적이며, 보통은 코사인법칙으로 한 각의 $\cos$ 값을 먼저 구해 $\sin$을 얻은 뒤 $\dfrac{1}{2}ab\sin C$를 쓰는 편이 빠르다.

사각형의 넓이

1) 평행사변형의 넓이

이웃하는 두 변의 길이가 $a,\,b$이고 그 끼인각의 크기가 $\theta$인 평행사변형의 넓이 $S$는

$$S = ab\sin\theta$$
2) 사각형의 넓이 (대각선 이용)

두 대각선의 길이가 $p,\,q$이고 두 대각선이 이루는 각의 크기가 $\theta$인 사각형 $\mathrm{ABCD}$의 넓이 $S$는

$$S = \dfrac{1}{2}pq\sin\theta$$
3) 일반 사각형 — 두 삼각형의 합

볼록사각형 $\mathrm{ABCD}$에서 한 대각선($\overline{\mathrm{AC}}$)을 그어 두 삼각형 $\mathrm{ABC},\,\mathrm{ACD}$로 나눈 뒤 각각의 넓이를 더한다. 두 삼각형이 한 변($\overline{\mathrm{AC}}$)을 공유하므로 코사인법칙으로 그 길이를 먼저 구한 뒤 사인공식을 적용한다.

$S = \triangle\mathrm{ABC} + \triangle\mathrm{ACD} = \dfrac{1}{2}\overline{\mathrm{AB}}\cdot\overline{\mathrm{BC}}\sin B + \dfrac{1}{2}\overline{\mathrm{AD}}\cdot\overline{\mathrm{DC}}\sin D$
대각의 합이 $180^\circ$인 원에 내접하는 사각형이면 $\sin B = \sin D$이므로 두 삼각형 넓이 식에서 $\sin$ 인수를 묶을 수 있다.

정답 및 해설

1.
[정답] (1) $\overline{\mathrm{AB}} = 10\sqrt{2}$   (2) $R = 10$

$B = 180^\circ - (105^\circ + 45^\circ) = 30^\circ.$

(1) $\overline{\mathrm{AC}} = b = 10$ ($\angle B$의 대변), $\overline{\mathrm{AB}} = c$ ($\angle C$의 대변)이므로 사인법칙 $\dfrac{c}{\sin C} = \dfrac{b}{\sin B}$에서

$c = \dfrac{b\sin C}{\sin B} = \dfrac{10\sin 45^\circ}{\sin 30^\circ} = \dfrac{10\cdot\frac{\sqrt{2}}{2}}{\frac{1}{2}} = 10\sqrt{2}.$

(2) 사인법칙 $\dfrac{b}{\sin B} = 2R$에서 $2R = \dfrac{10}{\sin 30^\circ} = 20,$  즉 $R = 10.$

2.
[정답] $B = 60^\circ,\ C = 90^\circ,\ c = 20$

사인법칙 $\dfrac{a}{\sin A} = \dfrac{b}{\sin B}$로 $\sin B$를 구한다.

$\sin B = \dfrac{b\sin A}{a} = \dfrac{10\sqrt{3}\cdot\frac{1}{2}}{10} = \dfrac{\sqrt{3}}{2}.$

$0^\circ \lt B \lt 90^\circ$ 조건에서 $B = 60^\circ.$

$C = 180^\circ - (30^\circ + 60^\circ) = 90^\circ,$  즉 $\mathrm{ABC}$는 직각삼각형.

$c = \dfrac{a\sin C}{\sin A} = \dfrac{10\cdot 1}{1/2} = 20.$

3.
[정답] $c = 6$

$A + B + C = 180^\circ$이므로 $A + B = 180^\circ - C,$  즉 $\sin(A+B) = \sin C.$

주어진 식에 대입 : $10\sin^2 C = 9 \Rightarrow \sin C = \dfrac{3}{\sqrt{10}}\;(\sin C \gt 0).$

사인법칙 $c = 2R\sin C = 2\sqrt{10}\cdot\dfrac{3}{\sqrt{10}} = 6.$

4.
[정답] (1) $\cos\theta = \dfrac{13}{16}$   (2) $\sin\theta = \dfrac{\sqrt{87}}{16}$

(1) 코사인법칙 $\overline{\mathrm{BC}}^2 = \overline{\mathrm{AB}}^2 + \overline{\mathrm{AC}}^2 - 2\cdot\overline{\mathrm{AB}}\cdot\overline{\mathrm{AC}}\cos\theta.$

$(2\sqrt{7})^2 = 4^2 + 8^2 - 2\cdot 4\cdot 8\cos\theta$ $\Rightarrow 28 = 80 - 64\cos\theta,$  즉 $\cos\theta = \dfrac{52}{64} = \dfrac{13}{16}.$

(2) $\sin^2\theta = 1 - \cos^2\theta = 1 - \dfrac{169}{256} = \dfrac{87}{256}.$

$0^\circ\lt\theta\lt 180^\circ$이므로 $\sin\theta\gt 0,$ $\sin\theta = \dfrac{\sqrt{87}}{16}.$

5.
[정답] (1) 풀이 참조   (2) $a = b$인 이등변삼각형

(1) 사인법칙·코사인법칙으로 모든 각을 변으로 표현. $\sin A = \dfrac{a}{2R},\ \sin B = \dfrac{b}{2R},\ \cos C = \dfrac{a^2+b^2-c^2}{2ab}$ 대입.

$a^2 = b^2 + ab\cdot\dfrac{a^2+b^2-c^2}{2ab} \Rightarrow 2a^2 = 2b^2 + a^2 + b^2 - c^2$
$\Rightarrow a^2 + c^2 = 3b^2.$

(2) $a\cos B = b\cos A$에 코사인 변형 $\cos B = \dfrac{c^2+a^2-b^2}{2ca},\ \cos A = \dfrac{b^2+c^2-a^2}{2bc}$ 대입.

6.
[정답] (1) $\cos B = \dfrac{7\sqrt{3}}{20}$   (2) $\overline{\mathrm{AC}} = \dfrac{2\sqrt{70}}{5}$

원에 내접하는 사각형이므로 $B + D = 180^\circ,$  즉 $\cos D = -\cos B.$

(1) 대각선 $\overline{\mathrm{AC}}$를 그어 두 삼각형 $\mathrm{ABC},\,\mathrm{ACD}$에 코사인법칙을 각각 적용.

$\overline{\mathrm{AC}}^2 = (2\sqrt{3})^2 + 4^2 - 2\cdot 2\sqrt{3}\cdot 4\cos B = 28 - 16\sqrt{3}\cos B.$

$\overline{\mathrm{AC}}^2 = (\sqrt{3})^2 + 2^2 - 2\cdot\sqrt{3}\cdot 2\cos D = 7 + 4\sqrt{3}\cos B.$

두 식의 우변이 같으므로 $28 - 16\sqrt{3}\cos B = 7 + 4\sqrt{3}\cos B$ $\Rightarrow 21 = 20\sqrt{3}\cos B,$  즉 $\cos B = \dfrac{21}{20\sqrt{3}} = \dfrac{7\sqrt{3}}{20}.$

(2) $\overline{\mathrm{AC}}^2 = 7 + 4\sqrt{3}\cdot\dfrac{7\sqrt{3}}{20} = 7 + \dfrac{84}{20} = \dfrac{56}{5}.$

$\therefore \overline{\mathrm{AC}} = \sqrt{\dfrac{56}{5}} = \dfrac{2\sqrt{70}}{5}.$

7.
[정답] $6\sqrt{7}\,\mathrm{cm}$

전개도 이용. 밑면 반지름 $r = 4,$ 모선 $\ell = 12$이므로 옆면을 펼치면 반지름 $12,$ 호의 길이 $2\pi\cdot 4 = 8\pi$인 부채꼴.

중심각 $\alpha = \dfrac{8\pi}{12} = \dfrac{2\pi}{3}\;(\text{rad}) = 120^\circ.$

$\mathrm{P}$를 펴서 부채꼴의 두 끝 위치 $\mathrm{P},\,\mathrm{P}'$에 두고 $\mathrm{Q}$는 모선 $\overline{\mathrm{OP}'}$의 중점이므로 $\overline{\mathrm{OQ}} = 6,\ \overline{\mathrm{OP}} = 12,\ \angle\mathrm{POQ} = 120^\circ.$

코사인법칙으로 $\overline{\mathrm{PQ}}^2 = 12^2 + 6^2 - 2\cdot 12\cdot 6\cos 120^\circ = 144 + 36 + 72 = 252.$

$\therefore \overline{\mathrm{PQ}} = \sqrt{252} = 6\sqrt{7}\,\mathrm{cm}.$

8.
[정답] (1) $a = 2\sqrt{21}$   (2) $S = 20\sqrt{3}$ 또는 $5\sqrt{3}$

(1) $S = \dfrac{1}{2}bc\sin A$에서 $20\sqrt{3} = \dfrac{1}{2}\cdot 8\cdot 10\sin A = 40\sin A,$  즉 $\sin A = \dfrac{\sqrt{3}}{2}.$

$0^\circ\lt A\lt 90^\circ$이므로 $A = 60^\circ,\ \cos A = \dfrac{1}{2}.$

코사인법칙 $a^2 = b^2 + c^2 - 2bc\cos A = 64 + 100 - 80 = 84,$  즉 $a = 2\sqrt{21}.$

(2) 코사인법칙 $a^2 = b^2 + c^2 - 2bc\cos A \Rightarrow 84 = 100 + c^2 - 10c$ $\Rightarrow c^2 - 10c + 16 = 0$ $\Rightarrow c = 2,\ 8.$

$S = \dfrac{1}{2}bc\sin A = \dfrac{5\sqrt{3}}{2}c.$ $c = 8$일 때 $S = 20\sqrt{3},$ $c = 2$일 때 $S = 5\sqrt{3}.$

9.
[정답] (1) $S = \dfrac{21\sqrt{11}}{4}$   (2) $R = \dfrac{15\sqrt{11}}{11}$   (3) $r = \dfrac{\sqrt{11}}{2}$

(1) 헤론의 공식. $s = \dfrac{5+7+9}{2} = \dfrac{21}{2}.$

$S = \sqrt{s(s-a)(s-b)(s-c)}$ $= \sqrt{\dfrac{21}{2}\cdot\dfrac{11}{2}\cdot\dfrac{7}{2}\cdot\dfrac{3}{2}}$ $= \sqrt{\dfrac{21\cdot 7\cdot 3\cdot 11}{16}} = \dfrac{21\sqrt{11}}{4}.$

(2) $S = \dfrac{abc}{4R}$에서 $R = \dfrac{abc}{4S} = \dfrac{5\cdot 7\cdot 9}{4\cdot\frac{21\sqrt{11}}{4}} = \dfrac{315}{21\sqrt{11}} = \dfrac{15}{\sqrt{11}} = \dfrac{15\sqrt{11}}{11}.$

(3) $S = \dfrac{1}{2}r(a+b+c) = \dfrac{r}{2}\cdot 21$에서 $r = \dfrac{2S}{21} = \dfrac{2}{21}\cdot\dfrac{21\sqrt{11}}{4} = \dfrac{\sqrt{11}}{2}.$

10.
[정답] (1) $xy = 4$   (2) $\overline{\mathrm{PQ}}_{\min} = 2$

(1) 두 삼각형 $\mathrm{APQ},\,\mathrm{ABC}$가 $\angle A$를 공유.

$\dfrac{\triangle\mathrm{APQ}}{\triangle\mathrm{ABC}} = \dfrac{\frac{1}{2}xy\sin 60^\circ}{\frac{1}{2}\cdot 8\cdot 3\sin 60^\circ} = \dfrac{xy}{24} = \dfrac{1}{6}.$

$\therefore xy = 4.$

(2) 코사인법칙 : $\overline{\mathrm{PQ}}^2 = x^2 + y^2 - 2xy\cos 60^\circ = x^2 + y^2 - xy.$

$xy = 4$ 대입 : $\overline{\mathrm{PQ}}^2 = x^2 + y^2 - 4.$

산술·기하평균 : $x^2 + y^2 \ge 2xy = 8$ (등호 $x = y = 2$일 때).

$\therefore \overline{\mathrm{PQ}}^2 \ge 8 - 4 = 4,\ \overline{\mathrm{PQ}}_{\min} = 2.$

11.
[정답] (1) $S = 28\sqrt{3}$   (2) $\overline{\mathrm{BD}} = 3$

(1) 평행사변형 $\mathrm{ABCD}$에서 삼각형 $\mathrm{ABC}$에 코사인법칙.

$13^2 = 7^2 + 8^2 - 2\cdot 7\cdot 8\cos B$ $\Rightarrow 169 = 113 - 112\cos B$ $\Rightarrow \cos B = -\dfrac{1}{2}.$

$90^\circ\lt B\lt 180^\circ$이므로 $B = 120^\circ,\ \sin B = \dfrac{\sqrt{3}}{2}.$

평행사변형 넓이 $S = \overline{\mathrm{AB}}\cdot\overline{\mathrm{BC}}\sin B = 7\cdot 8\cdot\dfrac{\sqrt{3}}{2} = 28\sqrt{3}.$

(2) 두 대각선 공식 $S = \dfrac{1}{2}\overline{\mathrm{AC}}\cdot\overline{\mathrm{BD}}\sin\theta$ 적용.

$6\sqrt{3} = \dfrac{1}{2}\cdot 8\cdot\overline{\mathrm{BD}}\cdot\sin 120^\circ = 2\sqrt{3}\,\overline{\mathrm{BD}}.$

$\therefore \overline{\mathrm{BD}} = 3.$

12.
[정답] $S = 10\sqrt{3}$

$B + D = 60^\circ + 120^\circ = 180^\circ$이므로 사각형 $\mathrm{ABCD}$는 원에 내접.

대각선 $\overline{\mathrm{AC}}$에 대해 두 삼각형 $\mathrm{ABC},\,\mathrm{ACD}$로 분할.

$\triangle\mathrm{ABC} = \dfrac{1}{2}\cdot 7\cdot 4\sin 60^\circ = 14\cdot\dfrac{\sqrt{3}}{2} = 7\sqrt{3}.$

$\overline{\mathrm{AC}}$를 코사인법칙으로 : $\overline{\mathrm{AC}}^2 = 49 + 16 - 56\cos 60^\circ = 65 - 28 = 37.$

$\triangle\mathrm{ACD}$에서 $\overline{\mathrm{CD}} = 3,\ D = 120^\circ$이고 $\overline{\mathrm{DA}}$는 미지수. 코사인법칙 : $37 = \overline{\mathrm{DA}}^2 + 9 - 6\overline{\mathrm{DA}}\cos 120^\circ = \overline{\mathrm{DA}}^2 + 9 + 3\overline{\mathrm{DA}}$ $\Rightarrow \overline{\mathrm{DA}}^2 + 3\overline{\mathrm{DA}} - 28 = 0$ $\Rightarrow \overline{\mathrm{DA}} = \dfrac{-3+\sqrt{121}}{2} = 4.$

$\triangle\mathrm{ACD} = \dfrac{1}{2}\cdot 4\cdot 3\sin 120^\circ = 6\cdot\dfrac{\sqrt{3}}{2} = 3\sqrt{3}.$

$S = 7\sqrt{3} + 3\sqrt{3} = 10\sqrt{3}.$
